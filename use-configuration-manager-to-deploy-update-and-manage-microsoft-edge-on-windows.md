# <a name="use-configuration-manager-to-deploy-update-and-manage-microsoft-edge-on-windows"></a>Použití nástroje Configuration Manager k nasazení, aktualizaci a správě Microsoft Edge ve Windows

Z verze Configuration Manageru 1910 můžete nasadit Microsoft Edge (verze 77 nebo novější) prostřednictvím skriptu PowerShellu. Skript vypíná automatické aktualizace a umožňuje spravovat aktualizace prostřednictvím nástroje Configuration Manager. Další informace o tomto a dalších výhodách používání nástroje Configuration Manager naleznete v tématu [Microsoft Edge Management](https://docs.microsoft.com/mem/configmgr/apps/deploy-use/deploy-edge?).
