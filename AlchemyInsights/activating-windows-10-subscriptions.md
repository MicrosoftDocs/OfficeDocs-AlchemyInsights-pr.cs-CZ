# <a name="activating-windows-10-subscriptions"></a>Aktivace předplatných Windows 10

Microsoft 365 Business, Education a Enterprise poskytují jako součást služby předplatného digitální licence pro Windows 10. Abyste předplatné mohli aktivovat, musí vaše zařízení už mít aktivovaný systém Windows 10 Pro (1703 nebo novější). Úplný seznam požadavků na aktivaci předplatného Windows 10 najdete v tématu [Požadavky na aktivaci předplatného Windows 10](https://docs.microsoft.com/windows/deployment/windows-10-subscription-activation#requirements).
