# <a name="learn-about--the-mapping-between-microsoft-edge-legacy-policies-and-microsoft-edge-policies"></a>Další informace o mapování mezi staršími zásadami Microsoft Edge a zásadami Microsoft Edge

Zásady skupiny starší verze Microsoft Edge (verze 45 a starší) a nastavení správy mobilních zařízení (MDM) se mapou na příslušné zásady Microsoft Edge, verze 80. Další informace najdete v tématu Mapování zásad [Microsoft Edge starší verze k Microsoft Edge](https://docs.microsoft.com/deployedge/microsoft-edge-policy-map-legacy-to-newedge).
