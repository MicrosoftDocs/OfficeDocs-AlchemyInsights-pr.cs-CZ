# <a name="in-place-upgrade-with-configuration-manager-guide"></a>Místní upgrade pomocí Průvodce konfigurací Configuration Manageru

V místě upgradu se zachová všechna data, nastavení, aplikace a ovladače z existující verze operačního systému. Je navržená tak, aby byla velmi spolehlivá a aby se v případě výskytu problémů automaticky vrátil do předchozího operačního systému.

Při upgradu zařízení s Windows 7 a Windows 8,1 na nejnovější verzi Windows 10 použijte [Průvodce místní instalací](https://admin.microsoft.com/adminportal/home#/win10upgrade) . Pomocí poskytnutého skriptu zkontrolujete předpoklady a automaticky nakonfigurujte místní upgrade.
