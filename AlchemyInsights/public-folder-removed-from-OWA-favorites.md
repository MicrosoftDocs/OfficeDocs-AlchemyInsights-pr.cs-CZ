# <a name="public-folder-removed-from-outlook-on-the-web-owa-favorites"></a>Veřejná složka odebráná Outlook oblíbených položek na webu (OWA)

Existuje známý problém s odebráním veřejných složek z oblíbených položek v aplikaci OWA při každém odhlášení nebo aktualizaci obrazovky. Problém se brzy opraví. Do té doby znovu přidejte složku pro přístup z OWA.
