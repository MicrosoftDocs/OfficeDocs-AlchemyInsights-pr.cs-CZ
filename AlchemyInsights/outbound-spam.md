# <a name="outbound-spam"></a>Odchozí Nevyžádaná pošta

Pokud se u odchozích nevyžádaných zpráv jedná o falešné zprávy (dobré odchozí zprávami označené jako nevyžádaná pošta), otevřete lístek odborné pomoci s příklady zpráv.
