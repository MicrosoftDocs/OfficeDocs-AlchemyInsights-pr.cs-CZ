# <a name="application--integration"></a>Integrace aplikací

- [Integrace služby Azure Active Directory s aplikací Začínáme](https://docs.microsoft.com/azure/active-directory/manage-apps/plan-an-application-integration)  – shrnuje proces integrace aplikací se službou Azure Active Directory (AD).

- [Co je Správa aplikací?](https://docs.microsoft.com/azure/active-directory/manage-apps/what-is-application-management)  – Službu Azure AD je nutné nakonfigurovat tak, aby byla integrovaná s aplikací. Jinými slovy, Azure AD potřebuje vědět, co aplikace používají jako systém identity. Proces udržení služby Azure AD s těmito aplikacemi a postupem jejich zpracování se nazývá Správa aplikací.

- [Kurzy pro integraci aplikací s Azure Active Directory](https://docs.microsoft.com/azure/active-directory/saas-apps/tutorial-list)  – pro účely integrace aplikací se službou Azure AD jsme vyvinuli kolekci výukových programů, které vás provedou konfigurací.
