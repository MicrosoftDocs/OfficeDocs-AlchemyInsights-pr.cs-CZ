# <a name="skype-icon-missing"></a>Chybí ikona Skype

Dočasně jsme odebrali ikonu Skype z Outlook.com jako součást aktualizace návrhu pro aktualizaci a funkce. Vážíme si trpělivosti, abychom vám připravili nové a vylepšené možnosti.

Skype můžete používat i v prohlížeči na adrese [Web.Skype.com](https://web.skype.com/).
