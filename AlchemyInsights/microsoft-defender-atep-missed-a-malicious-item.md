# <a name="microsoft-defender-atp-missed-a-malicious-item"></a>Ochrana ATP v programu Microsoft Defender vynechala škodlivou položku

- Odešlete škodlivý soubor společnosti Microsoft k analýze na [portálu Microsoft Defender Response Portal](https://www.microsoft.com/wdsi/filesubmission/). 
- Pokud jste přihlášeni, vyhledejte historii odeslání souboru na [portálu Microsoft Defender Response Portal](https://www.microsoft.com/wdsi/submissionhistory).
- Další informace naleznete v tématu [Odeslání souborů k analýze](https://docs.microsoft.com/windows/security/threat-protection/intelligence/submission-guide).
