# <a name="getting-an-exchange-hybrid-key"></a>Klíč pro výměnu hybridní získávání

Chcete-li připojit místní organizace Exchange online v Office 365, musíte nastavit alespoň jeden server Exchange v organizaci v prostorách. Tento server bude použit k vytvoření hybridní nasazení, který se připojuje k serveru Exchange Online organizaci používající server Exchange v prostorách. Chcete-li získat klíč hybrid pro tento server, postupujte na [Průvodce distribuce klíčů produktu Exchange hybridní](http://aka.ms/hybridkey).
