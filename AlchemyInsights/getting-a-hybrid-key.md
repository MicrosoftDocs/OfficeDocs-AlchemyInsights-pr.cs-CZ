# <a name="getting-an-exchange-hybrid-key"></a>Získání hybridního klíče Exchange

Pokud chcete místní organizaci připojit k Exchangi Online v Microsoft 365, musíte nastavit aspoň jeden server Exchange v místní organizaci. Tento server se používá k vytvoření hybridního nasazení, které připojuje vaši místní organizaci Exchange k Exchangi Online. Pokud chcete získat hybridní klíč pro tento server, postupujte podle pokynů v [Průvodci distribucí hybridního kódu Product Key pro Exchange](https://aka.ms/hybridkey).
