# <a name="onboarding-microsoft-defender-atp"></a>Připojení programu Microsoft Defender ATP

Informace o prostředcích k připojení ochrany ATP v Microsoft Defenderu pro windows 10, servery Windows a počítače Mac a Linux najdete v následujících tématech: 

- **Počítače s Windows 10** V závislosti na vašich obchodních požadavcích si můžete vybrat z možností pro zařízení s Windows 10. Další informace najdete [v tématu Nástroje a metody zařazení pro zařízení s Windows 10](https://docs.microsoft.com/windows/security/threat-protection/microsoft-defender-atp/configure-endpoints). 

- **Servery windows** Informace o připojení k programu Microsoft Defender ATP pro vaši verzi systému Windows Server naleznete [v tématu Integrované servery služby Ochrany ATP programu Microsoft Defender](https://docs.microsoft.com/windows/security/threat-protection/microsoft-defender-atp/configure-server-endpoints).

- **Koncové body jiného než systému Windows**  Před nástupem do systému jiného výrobce než Windows musíte splňovat systémové požadavky pro [Mac](https://docs.microsoft.com/windows/security/threat-protection/microsoft-defender-atp/microsoft-defender-atp-mac#system-requirements) nebo [Linux](https://docs.microsoft.com/windows/security/threat-protection/microsoft-defender-atp/microsoft-defender-atp-linux#system-requirements).

    Po potvrzení systémových požadavků si přečtěte informace [o tom, jak najdete integrované zařízení bez systému Windows](https://docs.microsoft.com/windows/security/threat-protection/microsoft-defender-atp/configure-endpoints-non-windows#onboarding-non-windows-machines).
