# <a name="site-is-not-available"></a>Web není k dispozici

Pokud web služby SharePoint nebo OneDrive není k dispozici pro více uživatelů, může být problém s dočasnou službu. Kontrola[Řídicí panel stavu služeb](https://admin.microsoft.com/AdminPortal/Home#/servicehealth) zobrazíte, pokud aplikace klienta klesá.
