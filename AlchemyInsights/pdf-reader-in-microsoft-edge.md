# <a name="pdf-reader-app-in-microsoft-edge"></a>Aplikace pro čtení PDF v Microsoft Edge

Oblíbenost souborů PDF zvýrazní potřebu spolehlivé, zabezpečené a výkonné čtečky PDF, kterou mohou podniková společnost přijmout. Tuto funkci nabízí čtečka PDF integrovaná do Microsoft Edge (verze 77 a novější verze). Funguje ve Windows a macOS a může se používat k otevření a anotování souborů PDF, které jsou místní, online nebo vložené na webových stránkách.

Další informace najdete v článku [o čtečce souborů PDF v Microsoft Edge.](https://docs.microsoft.com/deployedge/microsoft-edge-pdf)
