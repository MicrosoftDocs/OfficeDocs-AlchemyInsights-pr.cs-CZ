# <a name="change-the-domain-for-microsoft-365-group"></a>Změna domény pro skupinu Microsoft 365

Pokud chcete během vytváření změnit doménu skupiny Microsoft 365, postupujte podle pokynů v článku Volba domény, kterou chcete použít při [vytváření skupin Microsoft 365.](https://docs.microsoft.com/microsoft-365/admin/create-groups/choose-domain-to-create-groups)
  
Po vytvoření skupiny Microsoft 365 není momentálně možné doménu změnit. Jediným způsobem je vytvořit novou skupinu Microsoft 365 a v případě potřeby odstranit starou skupinu (další e-mailové adresy je možné přidat prostřednictvím Centra pro správu [Exchange).](https://outlook.office365.com/ecp.aspx)
