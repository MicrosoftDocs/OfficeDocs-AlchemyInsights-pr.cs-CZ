# <a name="get-the-a1-plan"></a>Dostanu plán A1

Pokud už má vaše instituce nárok na Microsoft 365 Education, můžete do **centra pro správu Microsoftu 365** přidat předplatné A1. 

1. Přejděte do části **Fakturace > [Produkty a služby](https://go.microsoft.com/fwlink/p/?linkid=868433)**.

2. Pomocí vyhledávacího pole hledejte "A1".

3. Vyberte počet uživatelů, kteří budou plán A1 používat.

Pokud vaše instituce zatím nastavila Microsoft 365 Education, ale vy chcete přidat Office 365 zdarma, můžete [začít](https://www.microsoft.com/education/products/office) tak, že zadáte svůj e-mail.
