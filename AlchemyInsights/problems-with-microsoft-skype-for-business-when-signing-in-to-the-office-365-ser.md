# <a name="problems-signing-in-to-microsoft-skype-for-business"></a>Problémy přihlášení ke společnosti Microsoft Skype pro firmy

Pokud máte potíže s přihlášením k Skype for Business, můžete řešit s podporou a Pomocník pro obnovení (SaRA). Tento nástroj umožňuje automaticky diagnostikovat a opravit řadu známé problémy týkající se přihlašování pro Skype pro firmy.
  
[Můžete se dozvědět více na tento odkaz](https://support.microsoft.com/help/4087361/troubleshooting-office-365-issues-signing-in-to-skype-for-business).
