# <a name="error-onedrive-cannot-connect-to-windows"></a>Chyba: funkce OneDrive se nemůže připojit k systému Windows.

Chcete-li tuto chybovou zprávu vyřešit, vyhledejte soubor cldflt. sys a zjistěte, zda je spuštěný, a pokud není, povolte jej. 
  
Další informace o povolení souboru cldflt. sys naleznete v tématu [OneDrive nelze se připojit k systému Windows při přístupu k souborům v OneDrive.](https://go.microsoft.com/fwlink/?Linkid=2031032)
