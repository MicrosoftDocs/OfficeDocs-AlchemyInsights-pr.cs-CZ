# <a name="fix-authentication-failed-errors-in-microsoft-flow"></a>Oprava chyb "Ověření se nezdařilo" v Microsoft Flow

Pokud se při pokusu o přihlášení k Microsoft Flow zobrazí chybová zpráva "Ověření se nezdařilo", je to proto, že služba Flow je ve službě Azure Active Directory zakázaná. Podrobnosti a postup opravy chyby najdete v tématu [Nemůžete se přihlásit k Microsoft Flow a zobrazí se chyba "Ověření se nezdařilo"](https://support.microsoft.com/help/4316891).
