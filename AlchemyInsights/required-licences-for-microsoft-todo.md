# <a name="required-licenses-for-microsoft-todo"></a>Požadované licence pro Microsoft ToDo

[Požadované licence pro Microsoft ToDo](https://support.office.com/article/381e9d1b-c500-49b5-973e-890fd86528d7.aspx)
  
Abyste mohli používat Microsoft ToDo, musí mít jeden z následujících licencí:
  
- Business Essentials

- Business Premium

- Microsoft 365 Firstline F1

- Plán Office 365 Enterprise

- Zabezpečme produktivní firmy

- Bezpečná produktivní organizace – E3

- Bezpečná produktivní organizace E5

- Plán Office 365 pro vzdělávací organizace

- Microsoft 365 a3

- Microsoft 365 a5

Kromě platné licence na produkt je potřeba, aby aplikace ToDo mohla synchronizovat uživatelská data:
  
- Exchange Online (plán 1)

- Exchange Online (plán 2)

- Základy systému Exchange

- Exchange Online

- Exchange Online POP

[Přiřazení licencí uživatelským účtům pomocí Office 365 PowerShellu](https://docs.microsoft.com/office365/enterprise/powershell/assign-licenses-to-user-accounts-with-office-365-powershell )
