# <a name="get-help-with-external-monitors"></a>Získání nápovědy k externím monitorům

Nastavení projektu můžete snadno otevřít stisknutím následujících kláves najednou:

- Klávesa Windows (tlačítko Start s logem Windows) + P
- V závislosti **na** vašich  předvolbách můžete zvolit **možnost** Duplikovat, Prodloužit nebo Druhou obrazovku.

Pokud máte problémy s externími monitory, zkuste tento postup řešení potíží:

- Odpojte kabel připojující počítač k monitoru a znovu ho připojte.
- Odpojte všechna zařízení, která nejsou nezbytná, od počítače (například adaptéry nebo doky).

- [Použití více monitorů ve Windows 10](https://support.microsoft.com/windows/how-to-use-multiple-monitors-in-windows-10-329c6962-5a4d-b481-7baa-bec9671f728a)
