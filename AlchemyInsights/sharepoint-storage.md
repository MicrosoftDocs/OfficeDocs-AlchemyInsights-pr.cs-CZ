# <a name="manage-your-sharepoint-online-storage"></a>Správa úložiště SharePointu Online

Informace o [výpočtu úložiště tenanta](https://docs.microsoft.com/office365/servicedescriptions/sharepoint-online-service-description/sharepoint-online-limits?redirectedfrom=MSDN#limits-by-plan) na základě vašeho plánu

U většiny organizací se úložiště SharePointu automaticky spravuje, takže nemusíte nastavovat limity pro jednotlivé kolekce webů.

Naučte se [Zobrazit celkový a dostupný prostor úložiště pro vaši organizaci](https://docs.microsoft.com/sharepoint/manage-site-collection-storage-limits).

Naučte se [Spravovat individuální limity úložiště kolekce webů](https://docs.microsoft.com/sharepoint/manage-site-collection-storage-limits#manage-individual-site-storage-limits).

Pokud váš web nemá dostatek místa nebo chcete zjistit, která knihovna nebo seznam používá prostor, přejděte na **Nastavení webu**  >  **Storage Metrics** (jsou vyžadována oprávnění správce kolekce webů).

Naučte se [sledovat limity úložiště webu pomocí PowerShellu](https://docs.microsoft.com/sharepoint/manage-site-collection-storage-limits#monitor-site-storage-limits-by-using-powershell).

Zjistěte, jak [si můžete koupit úložiště](https://docs.microsoft.com/microsoft-365/commerce/add-storage-space).
