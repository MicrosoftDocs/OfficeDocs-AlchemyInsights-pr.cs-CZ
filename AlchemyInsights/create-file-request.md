# <a name="how-to-create-a-file-request"></a>Jak vytvořit požadavek na soubor

Funkce žádosti o soubor na OneDrivu umožňuje vybrat složku, do které můžou ostatní nahrávat soubory pomocí odkazu, který jim pošlete. Uživatelé, od kterých požadujete soubory, nemohou zobrazit, co je ve složce – mohou do ní nahrávat pouze soubory.

[Vytvoření požadavku na soubor](https://support.office.com/article/create-a-file-request-f54aa7f8-2589-4421-b351-d415fc3b83af)
