# <a name="microsoft-defender-for-office-365-advisor"></a>Poradce Microsoft Defender pro Office 365 Advisor

[Nástroj Microsoft Defender pro Office 365 Advisor](https://go.microsoft.com/fwlink/?linkid=2146614) chrání vaši organizaci před škodlivými hrozbami, ke kterým může vaše prostředí dojít prostřednictvím e-mailových zpráv, odkazů a nástrojů pro spolupráci třetích stran. Tento průvodce vám poskytne zdroje a informace, které vám pomůžou připravovat a identifikovat plán aplikace Microsoft Defender pro potřeby organizace.
