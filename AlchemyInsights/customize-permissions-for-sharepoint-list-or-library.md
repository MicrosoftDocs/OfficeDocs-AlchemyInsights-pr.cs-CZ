# <a name="how-to-customize-permissions-for-a-sharepoint-list-or-library"></a>Přizpůsobení oprávnění pro seznam nebo knihovnu SharePointu

Jako správce nebo vlastník knihovny, seznamu nebo průzkumu můžete změnit oprávnění a umožnit jedné skupině uživatelů přístup k datům, která potřebují při omezování ostatních osob.
