# <a name="how-to-customize-permissions-for-a-sharepoint-list-or-library"></a>Jak přizpůsobit oprávnění pro sharepointový seznam nebo knihovnu

Jako správce nebo vlastník knihovny, seznamu nebo průzkumu můžete změnit oprávnění tak, aby jedna skupina lidí mohla přistupovat k datům, která potřebují, a zároveň omezit ostatní.

[Přizpůsobení oprávnění pro sharepointový seznam nebo knihovnu](https://support.office.com/article/customize-permissions-for-a-sharepoint-list-or-library-02d770f3-59eb-4910-a608-5f84cc297782)
