# <a name="change-the-domain-for-microsoft-365-group"></a>Změna domény pro skupinu Microsoft 365

Pokud chcete během vytváření změnit doménu pro skupinu Microsoft 365, postupujte podle pokynů v [tématu Výběr domény, kterou chcete použít při vytváření skupin systému microsoft 365](https://docs.microsoft.com/microsoft-365/admin/create-groups/choose-domain-to-create-groups).

Po vytvoření skupiny Microsoft 365 už není možné doménu změnit. Jediným způsobem je vytvoření nové skupiny Microsoft 365 a odstranění staré v případě potřeby (další e-mailové adresy lze přidat prostřednictvím [centra pro správu Exchange](https://outlook.office365.com/ecp)).
