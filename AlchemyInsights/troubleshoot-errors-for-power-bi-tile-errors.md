# <a name="troubleshooting-refresh-scenarios---tile-errors"></a>Odstraňování potíží scénáře aktualizace - chyby vedle sebe

Pokud dochází k chybám aktualizace dlaždice v rámci služby Power BI, počkejte asi 10-15 minut pro aktualizace se projeví v dashboard dlaždice. Pokud se stále nezobrazuje, znovu připnout vizualizace do řídicího panelu.

Další informace:[https://docs.microsoft.com/power-bi/refresh-troubleshooting-refresh-scenarios](https://docs.microsoft.com/power-bi/refresh-troubleshooting-refresh-scenarios)
