# <a name="deploy-and-update-microsoft-365-apps-with-configuration-manager-advisor"></a>Nasazení a aktualizace aplikací Microsoft 365 pomocí Správce konfigurace poradce

V organizacích používajících Správce konfigurace můžete pomocí poradce pro nasazení Správce konfigurace aktualizaci [aplikací Microsoft 365](https://go.microsoft.com/fwlink/?linkid=2146549) vygenerovat skript, který automaticky nakonfiguruje nasazení aplikací Microsoft 365 pomocí osvědčených postupů doporučených inženýry FastTracku.

V tomto průvodci můžete vytvářet skupiny nasazení, přizpůsobovat aplikace a funkce Office, konfigurovat dynamické nebo štíhlé instalace a pak spustit skript k vytvoření aplikací, pravidel automatického nasazení a kolekcí zařízení, které potřebujete k cíli nasazení.
