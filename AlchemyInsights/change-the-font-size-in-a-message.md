# <a name="change-the-font-size-in-a-message"></a>Změna velikosti písma ve zprávě

1. Pokud jste zprávu už napsali, vyberte text.
2. Vyberte ikonu  **formátu** pod zprávou (je to první ikona).
3. Vyberte ikonu  **velikosti písma**  nad zprávou (Jedná se o sedmou ikonu).
4. Vyberte velikost.
