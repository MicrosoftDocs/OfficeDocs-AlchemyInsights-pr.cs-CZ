# <a name="microsoft-365-urls-and-ip-address-ranges"></a>Adresy URL a rozsahy IP adres v Microsoftu 365

Microsoft 365 vyžaduje připojení k internetu prostřednictvím určitých IP adres, které se nazývají *koncové body*.
Seznam koncových bodů, které musí být dosažitelné, najdete v tématu Adresy URL a rozsahy IP adres v [Microsoftu 365.](https://docs.microsoft.com/office365/enterprise/urls-and-ip-address-ranges) 

Webová adresa IP a webová adresa URL microsoftu 365 vám pomůže lépe identifikovat a odlišit síťový provoz Microsoftu 365, což usnadňuje vyhodnocování, konfiguraci a aktuální informace o změnách. Podívejte se na webovou službu IP adresa a [adresa URL Microsoftu 365.](https://docs.microsoft.com/office365/enterprise/office-365-ip-web-service)
