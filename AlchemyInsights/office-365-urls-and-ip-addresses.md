# <a name="microsoft-365-urls-and-ip-address-ranges"></a>Adresy URL a rozsahy IP adres od Microsoftu 365

Microsoft 365 vyžaduje připojení k Internetu prostřednictvím konkrétních IP adres nazývaných *koncové body*.
Seznam koncových bodů, které je třeba k zastihnout, najdete v [tématu Adresy URL a rozsahy IP adres microsoftu 365.](https://docs.microsoft.com/office365/enterprise/urls-and-ip-address-ranges) 

Webová služba Ip adresy a adresy URL společnosti Microsoft 365 vám pomůže lépe identifikovat a rozlišit síťový provoz microsoftu 365, což usnadňuje vyhodnocení, konfiguraci a sledování se změnami. Viz [Webová služba IP adresy a adresy URL Office 365](https://docs.microsoft.com/office365/enterprise/office-365-ip-web-service).
