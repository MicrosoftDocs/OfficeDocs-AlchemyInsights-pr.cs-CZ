# <a name="recent-data-exposure"></a>Nedávná expozice údajů

Společnost Microsoft byla informována výzkumníkem z jiného výrobce, že databáze hostující redigovaná data podpory společnosti Microsoft byla přístupná bez ověřování. 31. prosince 2019 společnost Microsoft zabezpečila databázi.

Zavázali jsme se k ochraně osobních údajů a bezpečnosti našich zákazníků a podnikáme kroky, abychom zabránili budoucím unechům tohoto problému. Zahájili jsme oznámení zákazníkům, jejichž data byla přítomna v této redigované databázi. Další informace naleznete v [informacích o povědomí o zákaznících týkajících se nedávné expozice informací](https://aka.ms/privacyinfo).
