# <a name="recent-data-exposure"></a>Nedávná expozice dat

Společnost Microsoft byla informována společností prosearcher, že databáze hostující data podpory Microsoftu byla přístupná bez ověřování. Společnost Microsoft databázi zabezpečila 31. prosince 2019.

Jsme zaznamenali ochranu osobních údajů a zabezpečení našich zákazníků a převzali na to, aby se zabránilo budoucímu opakování tohoto problému. Zahájili jsme oznámení pro zákazníky, jejichž data byla v této redigováné databázi obsažená. Další informace najdete v [informacích o povědomí zákazníků týkajících se nedávného ohrožení informací](https://aka.ms/privacyinfo).
