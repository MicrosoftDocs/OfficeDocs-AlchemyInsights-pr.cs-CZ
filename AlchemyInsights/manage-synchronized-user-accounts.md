# <a name="manage-synchronized-user-accounts"></a>Správa synchronizovaných uživatelských účtů

Centrum pro správu Microsoftu 365 se obvykle používá k přidávání a změně uživatelských účtů. Však nelze použít, pokud sychronizace adresáře nebo saso jsou povoleny. Místo toho můžete spravovat uživatele pomocí prostředí Windows PowerShell, jak je popsáno v [nawebu Konfigurace vlastností uživatelského účtu pomocí prostředí Office 365 PowerShell](https://docs.microsoft.com/office365/enterprise/powershell/configure-user-account-properties-with-office-365-powershell ).
