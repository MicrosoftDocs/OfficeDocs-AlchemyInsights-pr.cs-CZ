# <a name="deploy-microsoft-365-apps"></a>Nasazení aplikací Microsoft 365

Při nasazování aplikací Microsoft 365 více koncovým uživatelům prostřednictvím nástroje Pro nasazení sady Office, nástroje Microsoft Endpoint Configuration Manager, jako součást bitové kopie disku operačního systému nebo ve scénáři sdíleného počítače pomocí systému Windows Server se službou Vzdálená plocha (RDS) nebo Terminálové služby zvažte použití podrobné příručky v **průvodci aplikací Microsoft 365 pro podnikové nasazení**.
  
Chcete-li začít, ujistěte se, že jste přihlášeni jako správce tenanta nebo globální správce, a klikněte sem: [https://aka.ms/proplusguidance](https://aka.ms/proplusguidance) .
