# <a name="install-office-with-the-microsoft-365-apps-deployment-advisor"></a>Instalace Office s poradcem pro nasazení Microsoft 365 Apps

Poradce [pro nasazení Microsoft 365 Apps](https://admin.microsoft.com/adminportal/home) vám pomůže získat zařízení uživatelů, na které běží nejnovější verze produktů Office, jako je Word, Excel, PowerPoint a OneNote.
  
Získáte pokyny k různým metodám nasazení, které zahrnují jednoduché možnosti vlastní instalace, a také informace o podnikových nasazeních používajících nástroje pro správu. Tyto pokyny vám pomůžou s posouzením vašeho prostředí, rozmyslíte si konkrétní požadavky na nasazení a implementujte potřebné nástroje podpory, které zajistí úspěšnou instalaci.
