# <a name="deploy-microsoft-365-apps"></a>Nasazení aplikací Microsoft 365

Pokud chcete nasadit aplikace pro Microsoft 365 více koncovými uživateli pomocí nástroje pro nasazení Office, Správce konfigurace Microsoft Endpoint, jako součást bitové kopie disku operačního systému nebo ve scénáři sdíleného počítače s Windows serverem se službou Vzdálená plocha (RDS) nebo s Terminálovou službou, zvažte použití podrobných pokynů v **Microsoft 365 Apps pro Enterprise Deployment Advisor**.
  
Abyste mohli začít, ujistěte se, že jste přihlášení jako správce tenanta nebo globální správce, a pak klikněte sem: [https://aka.ms/proplusguidance](https://aka.ms/proplusguidance) .
