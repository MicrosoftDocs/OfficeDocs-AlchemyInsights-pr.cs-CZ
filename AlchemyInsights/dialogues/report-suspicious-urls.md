# <a name="report-suspicious-urls"></a>Nahlášení podezřelých adres URL

Chcete-li odeslat podezřelé adresy URL, postupujte takto:

1. Do pole **Nové odeslání** zadejte úplnou adresu URL včetně protokolu (https://).
2. Pokud vyberete **Možnost Měla být blokována,** určete, zda jde o adresu URL **phishing** nebo **Malware.**
3. Vyberte **Odeslat.**
