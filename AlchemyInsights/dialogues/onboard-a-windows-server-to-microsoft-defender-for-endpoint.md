# <a name="onboard-a-windows-server-to-microsoft-defender-for-endpoint"></a>Připojení serveru Windows k programu Microsoft Defender pro koncový bod

Program Microsoft Defender pro koncový bod v operačním systému Windows Server poskytuje pokročilé funkce pro zjišťování a zkoumání útoků. Tyto funkce jsou poskytované bezproblémově prostřednictvím konzoly Centra zabezpečení v programu Microsoft Defender.

Program Defender pro koncový bod podporuje následující verze Windows Serveru:

- Windows Server 2008 R2 SP1
- Windows Server 2012 R2
- Windows Server 2016
- Windows Server (VYK) verze 1803 a novější
- Windows Server 2019 a novější
- Windows Server 2019 Core Edition

Další informace najdete v [části Onboard Windows servers to the Microsoft Defender for Endpoint service](https://go.microsoft.com/fwlink/?linkid=2143627).
