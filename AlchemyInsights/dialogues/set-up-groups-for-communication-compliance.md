# <a name="set-up-groups-for-communication-compliance"></a>Nastavení skupin pro dodržování předpisů při komunikaci

Tento krok je nepovinný. Pokud chcete začít pracovat jenom s několika uživateli a zjistit, jak dodržování předpisů v komunikaci funguje, můžete tento krok přeskočit.  
  
Když vytváříte zásady dodržování předpisů pro komunikaci, definujete, kdo má zkontrolovat jejich komunikaci a kdo bude provádět kontroly. Pomocí e-mailových adres můžete identifikovat jednotlivce nebo skupiny lidí. Pokud si chcete zjednodušit nastavení, můžete vytvořit skupiny pro uživatele, kteří mají s její komunikací přehled, a skupiny pro uživatele, kteří si sdělení prochová.  
  
Další informace najdete v tématu [Nastavení skupin pro dodržování předpisů při komunikaci.](https://go.microsoft.com/fwlink/?linkid=2129594)
