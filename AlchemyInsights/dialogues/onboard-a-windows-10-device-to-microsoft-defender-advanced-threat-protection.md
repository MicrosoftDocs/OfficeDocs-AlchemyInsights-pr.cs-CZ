# <a name="onboard-a-windows-10-device-to-microsoft-defender-advanced-threat-protection-mdatp"></a>Připojení zařízení s Windows 10 k rozšířené ochraně před internetových útoky v programu Microsoft Defender (MDATP)

MDATP podporuje následující nástroje a metody nasazení pro Windows 10:

- Zásady skupiny
- Microsoft Endpoint Configuration Manager
- Správa mobilních zařízení včetně Microsoft Intune
- Místní skripty

Další informace najdete v tématu Nástroje a metody [zboardování pro zařízení s Windows 10.](https://go.microsoft.com/fwlink/?linkid=2143460)
