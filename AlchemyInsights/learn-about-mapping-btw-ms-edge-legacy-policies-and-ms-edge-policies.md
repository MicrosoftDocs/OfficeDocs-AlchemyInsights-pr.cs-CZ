# <a name="learn-about-mapping-between-microsoft-edge-legacy-policies-and-microsoft-edge-policies"></a>Informace o mapování mezi zásadami starší verze Microsoft Edge a zásadami Microsoft Edge

Zásady skupiny pro Microsoft Edge starší (verze 45 a starší) a nastavení správy mobilních zařízení (MDM) se mapují na relevantní zásady pro Microsoft Edge verze 80. Další informace najdete v tématu [Microsoft Edge starším s mapováním zásad Microsoft Edge](https://go.microsoft.com/fwlink/?linkid=2141665).
