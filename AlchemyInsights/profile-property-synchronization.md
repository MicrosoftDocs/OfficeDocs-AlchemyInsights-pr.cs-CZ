# <a name="profile-property-synchronization"></a>Synchronizace vlastností profilu

Není možné mapovat další vlastnosti pro import adresáře AD pro synchronizaci z Azure Active Directory do aplikace profilů uživatelů. Import AD je omezen na předem nakonfigurovanou sadu vlastností, která zaručuje konzistentní výkon úlohy časovače.
  
[Další informace o vlastnostech profilu, které jsou synchronizovány](https://go.microsoft.com/fwlink/?linkid=875671)
