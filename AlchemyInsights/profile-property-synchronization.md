# <a name="profile-property-synchronization"></a>Synchronizace vlastností profilu

Není možné mapovat další vlastnosti pro import služby AD k synchronizaci ze služby Azure Active Directory do aplikace profilu uživatele. Import služby AD je omezen na předem nakonfigurovanou sadu vlastností, která zaručuje konzistentní výkon úlohy časovače.
  
[Další informace o vlastnostech profilu, které se synchronizují](https://go.microsoft.com/fwlink/?linkid=875671)
