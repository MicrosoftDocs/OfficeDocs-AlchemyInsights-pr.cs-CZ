# <a name="learn-about-microsoft-365-groups"></a>Další informace o skupinách Microsoft 365

Skupiny Microsoft 365 jsou služby, které používají již používané nástroje Office 365, takže můžete spolupracovat s ostatními členy týmu při psaní dokumentů, vytváření tabulek, práci na plánech projektu, plánování schůzek nebo odesílání e-mailů. Další informace najdete v článku [informace o skupinách Microsoft 365](https://support.office.com/article/b565caa1-5c40-40ef-9915-60fdb2d97fa2).
