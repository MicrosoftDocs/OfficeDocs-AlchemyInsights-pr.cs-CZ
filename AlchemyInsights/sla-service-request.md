# <a name="credit-request-due-to-a-service-outage"></a>Žádost o kredit v důsledku výpadku služby

Pokud chcete požádat o žádost o kredit kvůli výpadku služby, otevřete žádost o službu.

Na [stránce Stav služby](https://docs.microsoft.com/office365/enterprise/view-service-health) v centru pro správu Microsoft 365 se ujistěte, že máte ID incidentu.

Uvědomte si, že jakmile je vaše žádost zpracována, povede ověřením, aby sledovala dopad výpadku.
