# <a name="onedrive-common-issues-and-resolutions"></a>Běžné problémy a řešení OneDrivu

Tady jsou některá často požadovaná témata týkající se problémů s OneDrivem:

- **Nastavení synchronizace OneDrivu**: [Synchronizujte soubory OneDrivu pro firmy](https://go.microsoft.com/fwlink/?linkid=533375) nebo [synchronizujte sharepointové soubory](https://go.microsoft.com/fwlink/?linkid=871666) se synchronizačním klientem OneDrivu pro Windows.  Používáte Mac? Můžete [synchronizovat soubory s OneDrive](https://support.office.com/article/Sync-files-with-the-OneDrive-sync-client-on-Mac-OS-X-d11b9f29-00bb-4172-be39-997da46f913f) s ním také.

- **Poradce při potížích se synchronizací OneDrivu:** Pokud OneDrive nesynchronizuje soubory tak, jak očekáváte, můžete [řešit problémy se synchronizací OneDrivu](https://go.microsoft.com/fwlink/?linkid=866431) pro Windows. Používáte Mac? Můžete [také řešit problémy se synchronizací.](https://support.office.com/article/fix-onedrive-sync-problems-on-a-mac-af3012d7-13ec-4ac9-bbb1-ebcd2a0cd756)
- **Limity OneDrivu**: Přečtěte si o [omezeních a omezeních,](https://support.office.com/article/Invalid-file-names-and-file-types-in-OneDrive-OneDrive-for-Business-and-SharePoint-64883a5d-228e-48f5-b3d2-eb39e07630fa) která se vztahují na názvy souborů a typy souborů při synchronizaci s OneDrivem.
- **Úložiště OneDrivu**: Podívejte se, jak [změnit úložný prostor OneDrivu konkrétního uživatele](https://docs.microsoft.com/onedrive/change-user-storage) nebo [spravovat úložiště OneDrivu pro firmy](https://support.office.com/article/Manage-your-OneDrive-for-Business-storage-31519161-059C-4764-B6F8-F5CD29F7FE68).
- **Protokoly auditu**: Podívejte se, jak můžete [pomocí vyhledávání v protokolu auditu](https://docs.microsoft.com/microsoft-365/compliance/search-the-audit-log-in-security-and-compliance#search-the-audit-log) zkontrolovat aktivitu souborů, složek, uživatelů nebo oprávnění.
