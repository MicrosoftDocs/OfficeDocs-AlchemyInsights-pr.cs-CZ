# <a name="onedrive-common-issues-and-resolutions"></a>Běžné problémy a řešení pro OneDrive

Tady je několik často požadovaných témat týkajících se problémů s OneDrivem:

- **Nastavení synchronizace OneDrivu**: [synchronizace souborů OneDrivu pro firmy](https://go.microsoft.com/fwlink/?linkid=533375) nebo [synchronizace sharepointových souborů](https://go.microsoft.com/fwlink/?linkid=871666) s synchronizačním klientem OneDrivu pro Windows  Používáte Mac? Můžete taky [synchronizovat soubory s OneDrivem](https://support.office.com/article/Sync-files-with-the-OneDrive-sync-client-on-Mac-OS-X-d11b9f29-00bb-4172-be39-997da46f913f) .

- **Poradce při potížích s synchronizací OneDrivu**: Pokud OneDrive nesynchronizuje soubory očekávaným způsobem, můžete [řešit problémy se synchronizací OneDrivu](https://go.microsoft.com/fwlink/?linkid=866431) pro Windows. Používáte Mac? Můžete taky [řešit problémy se synchronizací](https://support.office.com/article/fix-onedrive-sync-problems-on-a-mac-af3012d7-13ec-4ac9-bbb1-ebcd2a0cd756) .
- **Limity pro OneDrive**: Zjistěte, jaká [omezení a omezení](https://support.office.com/article/Invalid-file-names-and-file-types-in-OneDrive-OneDrive-for-Business-and-SharePoint-64883a5d-228e-48f5-b3d2-eb39e07630fa) platí pro názvy souborů a typy souborů při synchronizaci s OneDrivem.
- **Úložiště OneDrive**: Přečtěte si téma [Změna prostoru úložiště OneDrivu určitého uživatele](https://docs.microsoft.com/onedrive/change-user-storage) nebo [Správa úložiště OneDrivu pro firmy](https://support.office.com/article/Manage-your-OneDrive-for-Business-storage-31519161-059C-4764-B6F8-F5CD29F7FE68).
- **Protokoly auditování**: Podívejte se, jak můžete [pomocí vyhledávání protokolu auditování](https://docs.microsoft.com/microsoft-365/compliance/search-the-audit-log-in-security-and-compliance#search-the-audit-log) zjistit aktivity souborů, složek, uživatelů nebo oprávnění.
