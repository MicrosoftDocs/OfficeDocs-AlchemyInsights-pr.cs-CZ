# <a name="whats-a-hub-site"></a>Co je centrální web?

Centrální web vám pomůže uspořádat weby a poskytnout jim společnou navigaci a branding. Globální správci nebo správci SharePointu budou moct vytvářet centrální weby a určovat, kteří uživatelé můžou přidružit své weby k webu centra. Další informace najdete v tématu [Weby centra SharePointu](https://go.microsoft.com/fwlink/?linkid=869388).
