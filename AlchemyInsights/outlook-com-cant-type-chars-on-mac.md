# <a name="how-to-type-special-characters-on-a-mac"></a>Jak psát speciální znaky na Macu

Chcete-li na Macu psát speciální znaky, otevřete nabídku **Speciální znaky** pomocí klávesové zkratky - **Command** + **Control** + **Space**.

Rychlá zkratka pro @ je také **ALT** + **64**.
