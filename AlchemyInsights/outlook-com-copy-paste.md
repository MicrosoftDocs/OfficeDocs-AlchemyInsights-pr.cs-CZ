# <a name="copy-and-paste-in-outlookcom"></a>Kopírování a vkládání v Outlook.com

V Outlook.com nejsou tlačítka pro kopírování ani vkládání, ale můžete použít klávesové zkratky:

**Na počítači:** CTRL+C pro kopírování, CTRL+X vyjmout a CTRL+V vložit.

**Na Macu:** CMD + C kopírovat, CMD + X vyjmout a CMD + V vložit.
