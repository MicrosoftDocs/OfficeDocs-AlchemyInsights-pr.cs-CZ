# <a name="copy-and-paste-in-outlookcom"></a>Kopírování a vkládání v Outlook.com

V Outlook.com nejsou tlačítka kopírovat ani vložit, ale můžete používat klávesové zkratky:

**Na počítači:** STISKNUTÍM CTRL + C kopírováním CTRL + X ho vyjmete a stisknutím CTRL + V ho vložíte.

**Na Macu:** CMD + C ke kopírování, CMD + X k vyjmutí a vložením příkazů CMD + V.
