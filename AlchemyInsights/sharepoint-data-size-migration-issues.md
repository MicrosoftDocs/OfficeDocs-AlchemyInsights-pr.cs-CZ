# <a name="issues-while-migrating-data-to-sharepoint-online"></a>Problémy při migraci dat do aplikace SharePoint Online

Zdá se, že přenášíte 100 TB dat na serveru SharePoint Online.

Kliknutím na tlačítko **Nová žádost o službu** telefon/email níže a ponechat jako migrace SharePoint více než 100 TB v textovém poli dotazu.

Po odeslání lístku aktualizujte jej s následujícími informacemi: 

- Název společnosti a adresa URL klienta
- Odhadovaná velikost migrace
- Pokud byste chtěli zahájit a dokončit migraci odhad
- Popsat, kde provádíte migraci obsahu, například serveru SharePoint Server, pole, GDrive, sdílených souborů, atd.
