# <a name="external-users-cant-send-email-to-microsoft-365-group"></a>Externí uživatelé nemohou odesílat e-maily do skupiny Microsoft 365

Pomocí těchto kroků povolte externím uživatelům e-mailem vaši skupinu Microsoft 365:

1. Přihlaste se do centra pro [správu](https://admin.microsoft.com/)  >  **Groups**  >  **Skupiny skupin**.
2. Vyberte skupinu, kterou chcete spravovat, a otevřete podokno Nastavení.
3. Na kartě **Nastavení** vyberte **Povolit externím odesílatelům e-mailem tuto skupinu**.
4. Vyberte **Uložit**.</br>
    Vyčkejte až hodinu, než se nastavení projeví.
