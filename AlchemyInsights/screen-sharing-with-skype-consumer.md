# <a name="screen-sharing-with-skype-consumer-users"></a>Sdílení obrazovky s spotřebitelskými uživateli Skype

Obecné pokyny týkající se sdílení programu, získání informací o omezeních se sdílením a odstraňování problémů s prostředky naleznete [v recenzi vaší obrazovky v programu Skype for Business](https://support.office.com/article/Share-your-screen-in-Skype-for-Business-2d436dc9-d092-4ef1-83f1-dd9f7a7cd3fc).  

V současné době není Skype pro firemní uživatele schopen sdílet položky s spotřebitelskými uživateli Skype. [Klepnutím sem získáte](https://www.skypefeedback.com/forums/299913-generally-available/suggestions/12335259-enable-screen-sharing-to-consumer-skype-users) zpětnou vazbu týkající se povolení sdílení obrazovky se spotřebiteli uživatelů Skype.
