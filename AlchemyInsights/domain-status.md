# <a name="domain-status---no-services-selected"></a>Stav domény – nejsou vybrány žádné služby.

Stav domény je k dispozici po připojení domény přímo k Microsoft 365, jako je Exchange. Pokud to chcete udělat, přejděte do Centra Microsoft 365 správy a vyberte **Záznamy DNS**  >  **Spravovat DNS**.

Pokud nemůžete doménu připojit přímo ke službám Microsoft 365, stav domény není dostupný. Pokud například používáte vlastní směrování DNS pro e-mail, externí služby filtrování spamu nebo používáte místní Exchange i Exchange Online (také se nazývá hybridní nasazení).
