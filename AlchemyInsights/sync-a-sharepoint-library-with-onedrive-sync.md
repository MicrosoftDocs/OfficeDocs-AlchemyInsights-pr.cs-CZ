# <a name="sync-a-sharepoint-library-with-onedrive-sync"></a>Synchronizace sharepointové knihovny se synchronizací OneDrivu

Chcete-li synchronizovat soubory služby SharePoint s místním pevným diskem, proveďte následující kroky:

1. Přejděte do knihovny dokumentů, kterou chcete synchronizovat s místním pevným diskem.
2. Vyberte tlačítko Synchronizovat z možností nabídky.
3. Postupujte podle kroku-za-krokem k dokončení synchronizace.
