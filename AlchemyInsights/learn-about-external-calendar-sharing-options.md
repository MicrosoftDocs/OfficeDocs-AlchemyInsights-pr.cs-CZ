# <a name="learn-about-external-calendar-sharing-options"></a>Informace o externích možnostech sdílení kalendáře

Pokud potřebujete naplánovat schůzky s lidmi mimo vaši organizaci, Microsoft 365 vám umožní zpřístupnit kalendáře "externím uživatelům", uživatelům, kteří potřebují zobrazit čas volného času, ale nemají uživatelské účty pro prostředí Microsoft 365. Další informace naleznete v tématu [Sdílení kalendářů s externími uživateli](https://docs.microsoft.com/office365/admin/manage/share-calendars-with-external-users).
