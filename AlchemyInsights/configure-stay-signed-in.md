# <a name="configure-stay-signed-in-for-accounts"></a>Konfigurace zůstat přihlášení pro účty

Funkce Zůstat přihlášení (KMSI)  zobrazí po úspěšném přihlášení uživatele výzvu Zůstat přihlášení? Pokud uživatel na tuto **výzvu** odpoví na ano, služba KMSI mu dá token trvalé aktualizace. 

Další informace o konfiguraci rozhraní KMSI najdete v tématu Konfigurace výzvy Zůstat přihlášení pro účty [Azure AD](/azure/active-directory/fundamentals/keep-me-signed-in).
