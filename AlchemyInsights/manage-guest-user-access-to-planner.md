# <a name="manage-guest-user-access-to-planner"></a>Správa přístupu uživatelů Guest k Planneru

Uživatelé typu Host můžou vytvářet a odstraňovat kontejnery a úkoly, upravovat pole úkolů a upravovat název plánu. Kromě toho se uživatelům typu host obdrží některá oznámení, která jsou spojená s vaším plánem. Tady je několik často požadovaných témat o hostujících uživatelích v Planneru:

- Buildy Planneru ve [skupině Microsoft 365 Groups Access](https://support.office.com/article/Adding-guests-to-Office-365-Groups-bfc7a840-868f-4fd6-a390-f347bf51aff6) 

- Přečtěte si další informace o tom, jak přidat uživatele typu Host do plánu, jak můžou uživatelé typu Host opustit plán, a taky to, co můžou hostující uživatelé dělat v Planneru v [Microsoft Planneru](https://support.office.com/article/Guest-access-in-Microsoft-Planner-cc5d7f96-dced-4da4-ab62-08c72d9759c6).
