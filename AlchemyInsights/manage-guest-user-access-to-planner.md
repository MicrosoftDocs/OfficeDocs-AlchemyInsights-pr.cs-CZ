# <a name="manage-guest-user-access-to-planner"></a>Správa přístupu uživatele typu Host k Planneru

Uživatelé typu Host mohou vytvářet a odstraňovat segmenty a úkoly, upravovat pole úkolů a upravovat název plánu. Uživatelé typu Host navíc obdrží také některá oznámení přidružená k vašemu plánu. Tady jsou některá často požadovaná témata o uživatelích typu Host v Planneru:

- Plánovač vychází z [funkce přístupu pro hosty skupin microsoftu 365 .](https://support.office.com/article/Adding-guests-to-Office-365-Groups-bfc7a840-868f-4fd6-a390-f347bf51aff6) 

- Přečtěte si další informace o tom, jak přidat uživatele typu Host do plánu, jak mohou uživatelé typu Host plán opustit a co mohou uživatelé typu Host dělat v aplikaci Planner in [Guest access v Microsoft Planneru](https://support.office.com/article/Guest-access-in-Microsoft-Planner-cc5d7f96-dced-4da4-ab62-08c72d9759c6).
