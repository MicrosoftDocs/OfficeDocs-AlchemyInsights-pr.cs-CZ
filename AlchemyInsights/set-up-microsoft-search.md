# <a name="set-up-microsoft-search"></a>Nastavení microsoft searchu

Microsoft Search‎ provides a user-friendly interface to search and find results in your organization across ‎Microsoft 365‎ services and applications to help you complete what you're working on. It is included at no additional cost with your eligible ‎Microsoft 365‎ subscription.

V [průvodci nastavením Microsoft Search můžete](https://go.microsoft.com/fwlink/?linkid=2156919) nakonfigurovat, jestli ho chcete pilotovat skupině uživatelů, nebo ji použít pro všechny uživatele ve vaší organizaci. Přiřadíte správcům vyhledávání a editorům vyhledávání a pak přizpůsobíte možnosti vyhledávání pro uživatele odpověďmi a dalšími možnostmi, jako je přidání rozšíření Bingu do Chromu nebo nastavení Bingu jako výchozího vyhledávacího stroje.
