# <a name="proxyaddress-incorrect"></a>ProxyAddress nesprávná

Když se objekt synchronizuje se službou Azure AD, hodnoty zadané v atributu proxyAddresses ve službě Active Directory jsou porovnány s pravidly Azure AD a pak je v Azure AD naplněn atribut proxyAddresses. Proto hodnoty atributu proxyAddresses pro objekt ve službě Active Directory nemusí být stejné jako hodnoty atributu proxyAddresses v Azure AD.
  
Další informace o tom, jak se ProxyAddress vyplní, najdete v tématu [jak se naplní atribut ProxyAddress v Azure AD](https://support.microsoft.com/help/3190357/how-the-proxyaddresses-attribute-is-populated-in-azure-ad).
