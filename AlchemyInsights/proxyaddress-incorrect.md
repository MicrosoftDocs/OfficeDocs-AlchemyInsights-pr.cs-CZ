# <a name="proxyaddress-incorrect"></a>ProxyAddress nesprávná

Při synchronizaci objektu do Služby Azure AD, hodnoty, které jsou zadány v atributu proxyAddresses ve službě Active Directory jsou porovnány s pravidly Azure AD a potom proxyAddresses atribut je naplněn v Azure AD. Proto hodnoty proxyAddresses atribut pro objekt ve službě Active Directory nemusí být stejné jako hodnoty proxyAddresses atribut v Azure AD.
  
Další informace o tom, jak je naplněna proxyadresa, najdete v [tématu Jak je vyplněn atribut proxyAddress ve službě Azure AD](https://support.microsoft.com/help/3190357/how-the-proxyaddresses-attribute-is-populated-in-azure-ad).
