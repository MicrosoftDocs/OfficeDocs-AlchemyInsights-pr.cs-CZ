# <a name="map-a-sharepoint-library-to-a-network-drive"></a>Mapovat síťovou jednotku do knihovny služby SharePoint

Knihovny jako síťovou jednotku mapování je dočasné a je podporován pouze pomocí aplikace Internet Explorer. Občas musí otevřít web služby SharePoint v aplikaci Internet Explorer a vyberte, chcete-li zabránit vypršení platnosti relace "Pobytu podepsané v". Místo toho [synchronizovat soubory služby SharePoint pomocí nového klienta synchronizace OneDrive](https://support.office.com/article/sync-sharepoint-files-with-the-new-onedrive-sync-client-6de9ede8-5b6e-4503-80b2-6190f3354a88) </a> který obsahuje [Soubory na vyžádání](https://support.office.com/article/learn-about-onedrive-files-on-demand-0e6860d3-d9f3-4971-b321-7092438fb38e). Přístup k všechny soubory v OneDrive bez použití místního úložného prostoru.

Pokud zvolíte mapovat jednotku namísto[nového klienta synchronizace OneDrive](https://support.office.com/article/sync-sharepoint-files-with-the-new-onedrive-sync-client-6de9ede8-5b6e-4503-80b2-6190f3354a88), zajistěte, aby že podle pokynů v níže uvedeném článku. 


**Konfigurace a odstraňování namapované síťové jednotky**


- [Konfigurace a odstraňování namapované síťové jednotky](https://support.office.com/article/troubleshoot-mapped-network-drives-that-connect-to-sharepoint-online-ef399c67-4578-4c3a-adbe-0b489084eabe?ui=en-US&amp;rs=en-US&amp;ad=US).

Poznámka: Pro aplikaci Internet Explorer 10 s Windows 8 nebo Windows 7, který se zobrazí "Přístup odepřen" nebo "Cesta není přístupná" při mapování jednotky, nainstalujte tuto opravu hotfix Chcete-li vyřešit tento problém. Přejděte k [chybě při otevření knihovny dokumentů služby SharePoint v Průzkumníku Windows nebo namapovat síťovou jednotku do knihovny po instalaci aplikace Internet Explorer 10](https://support.microsoft.com/help/2846960).
