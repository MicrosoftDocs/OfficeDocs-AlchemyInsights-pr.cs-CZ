# <a name="view-and-delete-browsing-history-in-microsoft-edge"></a>Zobrazení a odstranění historie procházení v Microsoft Edge

S vaším svolením si nový Microsoft Edge pamatuje vaše údaje o procházení, což vám pomůže vrátit se na oblíbený web nebo vyplnit formulář. V režimech InPrivate, Guest a dalších ukládá Microsoft Edge během normálního procházení méně dat, než ukládá.

Data procházení se ukládají na vašem zařízení. Pokud jste zapnuli synchronizaci, data se ukládají v cloudu a synchronizují se na všech zařízeních, kde jste přihlášení ke svému účtu Microsoft Edge.

Pokud chcete zobrazit historii procházení, vyberte Nastavení a **další možnosti ...**   >  **Historie**  >  **Správa historie** Historii procházení můžete kdykoli vymazat.
