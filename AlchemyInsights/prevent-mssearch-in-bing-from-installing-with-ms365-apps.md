# <a name="prevent-microsoft-search-in-bing-from-installing-with-microsoft-365-apps"></a>Zabránění instalaci Microsoft Search v Bingu s aplikacemi Microsoft 365

Pokud chcete službě Microsoft Search na pozadí v Bingu zabránit v instalaci s aplikacemi Microsoft 365 pro podniky, použijte jednu z následujících možností:

- Nástroj pro nasazení Office nebo Zásady skupiny
- Microsoft Endpoint Správce konfigurace (aktuální větev) nebo Microsoft Intune

Další informace najdete v tématu Jak zabránit instalaci služby Microsoft Search na pozadí v [Bingu s aplikacemi Microsoft 365 pro podniky](https://go.microsoft.com/fwlink/?linkid=2151946).
