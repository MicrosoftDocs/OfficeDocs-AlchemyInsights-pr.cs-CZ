# <a name="learn-about-site-compatibility-affecting-changes-coming-to-microsoft-edge"></a>Přečtěte si o změnách, které ovlivňují kompatibilitu webu, které přicházejí do Microsoft Edge.

Web se neustále vyvíjí, aby se zlepšilo uživatelské prostředí, zabezpečení a ochrana osobních údajů. Některé změny jsou dost významné, aby ovlivnily funkčnost existujících stránek. Tým Microsoft Edge aktualizuje stránky podle toho, jak se vyvíjí myšlení, časová osa se ztuhnou a oznamují [se změny.](https://go.microsoft.com/fwlink/?linkid=2135534)
