# <a name="hub-site-permissions"></a>Oprávnění webu centra

Správa oprávnění centrální lokality závisí na podkladovém typu webu. Pokud se jedná o týmový web připojený ke skupině, měli byste spravovat oprávnění prostřednictvím skupiny Microsoft 365. Pokud je to komunikační web, měli byste spravovat oprávnění prostřednictvím skupin SharePointu.

- [Oprávnění webu centra](https://docs.microsoft.com/sharepoint/modern-experience-sharing-permissions#hub-site-permissions)  
- [Weby plánovacího centra](https://docs.microsoft.com/sharepoint/planning-hub-sites)
