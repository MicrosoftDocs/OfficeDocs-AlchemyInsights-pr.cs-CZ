# <a name="use-microsoft-graph-api-to-manage-groups"></a>Použití rozhraní Microsoft Graph API ke správě skupin

Skupiny jsou kolekce uživatelů a další objekty zabezpečení, které sdílejí přístup k prostředkům ve službách Microsoftu nebo ve vaší aplikaci. Microsoft Graph nabízí rozhraní API, která můžete použít k vytváření a správě různých typů skupin a skupin podle vašeho scénáře. Všechny operace související se skupinou v Microsoft graphu vyžadují souhlas správce. Další informace najdete v článku [práce se skupinami v Microsoft graphu](https://docs.microsoft.com/graph/api/resources/groups-overview).
