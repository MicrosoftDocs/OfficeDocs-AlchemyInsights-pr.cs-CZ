# <a name="upload-process-overview-in-microsoft-stream"></a>Nahrání přehledu procesu v Microsoft streamu

Když video nahráváte, Microsoft Stream ho automaticky zakóduje do víc přenosových rychlostí, aby bylo možné streamovat pomocí datových proudů přizpůsobivého přenosu. Microsoft Stream optimalizuje přenos a kvalitu přehrávání videa na základě dostupné šířky pásma prohlížeče.

- [Nahrání přehledu procesu v Microsoft streamu](https://docs.microsoft.com/stream/upload-process-overview)
