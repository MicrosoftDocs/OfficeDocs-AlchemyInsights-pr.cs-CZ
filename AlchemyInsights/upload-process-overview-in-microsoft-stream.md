# <a name="upload-process-overview-in-microsoft-stream"></a>Přehled procesu nahrávání v Microsoft Streamu

Kdykoli video nahrajete, Microsoft Stream ho automaticky zakóduje do více datových toků, aby bylo možné video streamovat pomocí adaptivního streamování datového toku. Program Microsoft Stream optimalizuje datový tok a kvalitu přehrávaného videa na základě dostupné šířky pásma prohlížeče.

- [Přehled procesu nahrávání v Microsoft Streamu](https://docs.microsoft.com/stream/upload-process-overview)
