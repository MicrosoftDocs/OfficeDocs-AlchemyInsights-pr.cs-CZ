# <a name="setting-up-a-hybrid-environment"></a>Nastavení hybridního prostředí

Chcete-li uživatelům povolit bezproblémovou spolupráci bez ohledu na to, zda jsou jejich data uložena na místních serverech nebo v Microsoftu 365, možná budete muset nastavit hybridní prostředí. Chcete-li získat přehled o možnostech, podívejte se [na integraci Microsoftu 365 s místními prostředími](https://docs.microsoft.com/office365/enterprise/office-365-integration).
