# <a name="change-your-pin"></a>Změna kódu PIN

Outlook.com nepoužije PIN kód, ale můžete se k počítači přihlásit pomocí KÓDU PIN nebo Windows Hello. Změna PIN kódu ve Windows 10:

1. Vyberte tlačítko Start nebo stiskněte klávesu Windows na klávesnici a pak vyberte **Možnosti** > přihlášení**k účtům** > **Sign-in options**.
2. Vyberte tlačítko **Změnit** pod **kódem PIN**.
3. Zadejte aktuální PIN kód a pod ním zadejte a potvrďte nový PIN kód.

[Přečtěte si další informace o Windows Hello.](https://support.microsoft.com/help/17215/)
