# <a name="change-your-pin"></a>Změna PIN kódu

Outlook.com nepoužívá PIN kód, ale můžete se přihlásit do počítače pomocí PIN kódu nebo Windows Hello. Změna PIN kódu ve Windows 10:

1. Vyberte tlačítko Start nebo na klávesnici stiskněte klávesu Windows a **pak vyberte**  >  **Accounts**  >  **Možnosti přihlášení**k účtům.
2. V části **PIN**klikněte na tlačítko **změnit** .
3. Zadejte svůj aktuální PIN a pak zadejte a potvrďte nový PIN kód.

[Přečtěte si další informace o Windows Hello.](https://support.microsoft.com/help/17215/)
