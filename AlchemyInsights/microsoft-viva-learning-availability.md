# <a name="microsoft-viva-learning-availability"></a>Dostupnost služby Microsoft Viva Learning

Od února 2021 si zobrazíme náhled aplikace Viva Learning pro Teams s malým počtem zákazníků a partnerů a budeme očekávat, že ji později v tomto roce zvidime. Doporučujeme, abyste se [tady zaregistrovali,](https://aka.ms/VivaLearningSignup) abyste se dozvěděli další informace o výuce Viva Learning, včetně dalších podrobností o nových prostředích, která budou dostupná.

Další informace najdete v [článku o oznámení aplikace Microsoft Viva Learning ve verzi Private Preview.](https://techcommunity.microsoft.com/t5/microsoft-viva-blog/announcing-microsoft-viva-learning-private-preview/ba-p/2107023)
