# <a name="add-or-remove-an-email-address-for-a-user"></a>Přidání nebo odebrání e-mailové adresy uživatele

Chcete-li začít, [postupujte podle následujících pokynů](https://portal.office.com/AdminPortal/Home#/AssistedGuide/addemailoptions) ve středisku Microsoft 365 Admin Center.

 **Poznámka**: pro uživatele můžete mít více než jednu e-mailovou adresu, ale další *e-mailové aliasy* lze použít pouze pro příjem e-mailů. Pokud potřebujete odeslat e-mail s aliasem, vytvořte [sdílenou poštovní schránku](https://docs.microsoft.com/office365/admin/email/create-a-shared-mailbox). Tímto způsobem získáte alias pro odesílání a přijímání pošty.
