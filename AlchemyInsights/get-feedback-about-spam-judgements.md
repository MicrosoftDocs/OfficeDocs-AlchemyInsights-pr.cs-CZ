# <a name="new-feature-get-feedback-about-spam-judgments"></a>Nová funkce: získat názor soudních rozhodnutí nevyžádané pošty

Jako správce můžete nyní snadno nahlásit podezřelé e-mailové zprávy, URL a obsahu společnosti Microsoft v [Aplikaci Explorer odeslání](https://protection.office.com/reportsubmission). Dokonce můžete pomoci pochopit, proč byl nesprávně označena legitimní zpráva. Další informace naleznete v tématu tohoto [oznámení](https://techcommunity.microsoft.com/t5/Security-Privacy-and-Compliance/Empower-security-teams-to-easily-report-suspicious-emails-amp/ba-p/752622).

Aby bylo easer, můžete povolit uživatelům, aby nahlásit společnosti Microsoft [doplněk hlášení](https://appsource.microsoft.com/product/office/WA104381180?src=office&tab=Overview).
