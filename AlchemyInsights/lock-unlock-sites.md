# <a name="lock-and-unlock-sites"></a>Uzamknutí a odemknutí webů

Jako globální Microsoft 365 nebo SharePoint můžete zablokovat přístup k webu nebo nastavit web jen pro čtení pomocí Microsoft PowerShellu ke změně stavu uzamčení webu. Další informace najdete v tématu [Uzamknutí a odemknutí webů](/sharepoint/manage-lock-status).
