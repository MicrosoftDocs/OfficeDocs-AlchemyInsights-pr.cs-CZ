# <a name="public-folder-access-from-mobile-devices"></a>Přístup k veřejné složce z mobilních zařízení

Přístup k veřejným složkám nefunguje pomocí mobilních zařízení, pro přístup k veřejným složkám použijte OWA (Outlook na webu) nebo Outlook pro stolní počítače s Windows nebo Outlook pro Mac.

Pokud chcete mít podobné funkce jako veřejné složky a máte na mobilních zařízeních přístup k obsahu, přečtěte [si informace o skupinách Microsoft 365 a získejte](https://support.office.com/article/learn-about-office-365-groups-b565caa1-5c40-40ef-9915-60fdb2d97fa2) lepší možnosti.
