# <a name="live-events-in-microsoft-stream"></a>Živé události v Microsoft Streamu

Živé události můžete vytvářet pomocí služby Microsoft Stream v celé organizaci. Můžete plánovat, vytvářet a poskytovat živé události pro různé scénáře, jako jsou události v rámci celé společnosti, aktualizace vedení a další.

- [Živé události v Microsoft Streamu](https://docs.microsoft.com/stream/live-event-overview)
- [Vytvoření živé události v Microsoft Streamu](https://docs.microsoft.com/stream/live-create-event)
- [Začínáme s živými událostmi Microsoft Teams](https://support.office.com/article/get-started-with-microsoft-teams-live-events-d077fec2-a058-483e-9ab5-1494afda578a)
- [Konfigurace kodérů pro živé vysílání v Microsoft Streamu](https://docs.microsoft.com/stream/live-encoder-setup)
