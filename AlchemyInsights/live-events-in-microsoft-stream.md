# <a name="live-events-in-microsoft-stream"></a>Živé události v Microsoft streamu

Živé události můžete vytvářet pomocí Microsoft streamu v organizaci. Můžete plánovat, vytvářet a poskytovat živé události pro různé scénáře, jako jsou události pro celou organizaci, vedoucí organizace a další.

- [Vytvoření živé události v Microsoft streamu](https://docs.microsoft.com/stream/live-create-event)
- [Začínáme s událostmi služby Microsoft Teams Live](https://support.office.com/article/get-started-with-microsoft-teams-live-events-d077fec2-a058-483e-9ab5-1494afda578a)
- [Konfigurace kodérů pro živé datové proudy v Microsoft streamu](https://docs.microsoft.com/stream/live-encoder-setup)
