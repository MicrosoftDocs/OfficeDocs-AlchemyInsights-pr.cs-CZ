# <a name="enable-seamless-sso"></a>Umožňují bezproblémové SSO

Azure Active Directory (Azure AD) bezproblémové Single Sign-On (bezproblémové SSO) automaticky přihlásí uživatele při práci na podnikové stolní počítače, které jsou připojeny k podnikové síti.
  
[Postupujte podle kroků v tomto článku umožňuje bezproblémové SSO](https://docs.microsoft.com/azure/active-directory/connect/active-directory-aadconnect-sso-quick-start)
