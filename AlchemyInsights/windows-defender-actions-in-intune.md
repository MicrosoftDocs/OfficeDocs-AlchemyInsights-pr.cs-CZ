# <a name="windows-defender-actions-in-intune"></a>Akce programu Windows Defender v Intune

Intune se dá použít k aktivaci kontroly na vyžádání a aktualizace signatur virů pro Windows Defender na jednotlivých zařízeních.

Po úspěšném spuštění vzdálené akce se aktivita projeví v protokolu událostí v programu Windows Defender.

Zásady ochrany koncového bodu Windows umožňují další nastavení funkcí programu Windows Defender, která se mají vytvořit v Intune a použitá pro sady zařízení.

Další podrobnosti o aktivaci akcí v programu Windows Defender najdete v článku [Konfigurace a spuštění kontroly antivirové ochrany Microsoft Defenderu na vyžádání](https://docs.microsoft.com/windows/security/threat-protection/windows-defender-antivirus/run-scan-windows-defender-antivirus).
