# <a name="solutions-for-unsupported-operating-system"></a>Řešení pro "Nepodporovaný operační systém"



Při pokusu o instalaci sady Office 365, zobrazí se pravděpodobně chybová zpráva **nepodporovaný operační systém** Pokud zařízení nesplňuje [požadavky na systém](https://products.office.com/office-system-requirements).
  
Vyžaduje Office 365:
  
- PC: Windows 10, Windows 8.1, Windows 8, Windows 7 Service Pack 1, Windows Server 2016, Windows Server 2012 R2, Windows Server 2012 nebo Windows Server 2008 R2
    
    NEBO
    
- Mac OS X 10.10
    
Pokud máte tuto zprávu při pokusu o instalaci sady Office na mobilním zařízení nebo Chromebook, naleznete v článku [instalace a spuštění sady Microsoft Office na Chromebook](https://support.office.com/article/32f14a23-2c1a-4579-b973-d4b1d78561ad?wt.mc_id=Alchemy_ClientDIA).
