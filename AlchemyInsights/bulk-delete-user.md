# <a name="bulk-delete-user"></a>Uživatel hromadného odstranění

Můžete odebrat velký počet uživatelů ze skupiny pomocí souboru hodnot oddělených čárkami (CSV) pro hromadné odstranění uživatelů. Pokud chcete odstranit hromadné uživatele pomocí portálu Azure AD, přečtěte si přečtěte si informace [o hromadném odstranění uživatelů ve službě Azure Active Directory](https://docs.microsoft.com/azure/active-directory/users-groups-roles/users-bulk-delete).
