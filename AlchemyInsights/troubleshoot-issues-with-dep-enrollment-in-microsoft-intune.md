# <a name="troubleshoot-issues-with-dep-enrollment-in-microsoft-intune"></a>Odstraňování problémů s funkcí Zabránění spuštění dat pro zápis v Microsoft Intune

Zobrazit zdroje uvedené níže nyní vyřešit váš problém. 
  
1. Pokud je funkce Zabránění spuštění dat zařízení nelze zapsat a MFA (vícenásobné ověření) je povoleno, zakažte MFA. Aktuálně není podporována MFA pro zápis funkce Zabránění spuštění dat
    
2. Pomocí [Poradce při potížích s portál Intune](https://devicemanagement.microsoft.com/#blade/Microsoft_Intune_DeviceSettings/TroubleshootBlade) diagnostikovat a vyřešit běžné chyby zápisu. Přečtěte [dokument](https://docs.microsoft.com/en-us/intune/help-desk-operators) pro další podrobnosti. 
    
3. Zkontrolujte tyto dokumenty seznam běžné chyby, které brání každému zápisu a usnesení: [Poradce při potížích Průvodce](https://support.microsoft.com/en-us/help/4039809/troubleshooting-ios-device-enrollment-in-intune) a [Poradce při potížích dokument](https://docs.microsoft.com/en-us/intune-classic/troubleshoot/troubleshoot-device-enrollment-in-intune)
    
4. [Další informace o programu pro zápis zařízení](https://docs.microsoft.com/en-us/intune/device-enrollment-program-enroll-ios).
