# <a name="move-email-to-microsoft-365"></a>Přesunutí e-mailu do Microsoft 365

Máte několik možností pro migraci e-mailů do Microsoft 365. [Poradce pro migraci pošty v Microsoft 365](https://aka.ms/alchemyinsight-mailmigrationadvisor) vám pomůže vybrat pro vaše prostředí nejlepší, a taky vás provede migrací pošty.
  
Pokud si chcete přečíst informace o možnostech bez zahájení migrace, podívejte se na článek o tom, jak [vybrat cestu migrace](https://docs.microsoft.com/Exchange/mailbox-migration/decide-on-a-migration-path).

Další nápovědu k migraci můžete získat také na stránce [přijmout Microsoft 365 se FastTrack](https://www.microsoft.com/fasttrack/microsoft-365/office-365) .
