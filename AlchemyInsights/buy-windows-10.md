# <a name="how-to-buy-a-copy-of-windows-10"></a>Jak koupit kopii Windows 10

Informace o aktuálních cenách a o tom, jak koupit a stáhnout plnou verzi Windows 10, přejděte na [Microsoft Store](https://www.microsoft.com/store/b/windows).

**Poznámka:** Bezplatný Windows 10 prostřednictvím aplikace Získání Windows 10 (GWX) skončil 29. července 2016.

**Důležité:** Pokud používáte Windows 7, Microsoft vám doporučuje přejít na nový počítač s Windows 10, protože Windows 7 dosáhl konce životního cyklu podpory 14. ledna 2020.
