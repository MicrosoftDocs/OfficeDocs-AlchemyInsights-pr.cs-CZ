# <a name="create-custom-microsoft-surface-operating-system-images"></a>Vytvoření vlastních obrázků operačního systému Microsoft Surface

Pomocí nástroje Microsoft Surface Deployment Accelerator (SDA) můžete zjednodušit a automatizovat nasazení nového operačního systému, který je přizpůsobený vaší organizace a splňuje doporučení Microsoftu.

1. V GitHubu přejděte na SurfaceDeploymentAccelerator.

1. Vyberte **Kód a** pak vyberte Otevřít v **desktopové aplikaci GitHub nebo** Stáhnout **ZIP.**

1. Přečtěte si **README.md** souboru a zjistěte, jak skript funguje.

1. Upravte skript s odpovídajícími proměnnými pro vaše prostředí podle dokumentu v souboru Readme a pak ho před spuštěním v testovacím prostředí zkontrolujte.
