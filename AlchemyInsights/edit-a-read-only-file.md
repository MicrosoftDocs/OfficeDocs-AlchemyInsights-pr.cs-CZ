# <a name="edit-a-read-only-file"></a>Úprava souboru jen pro čtení

Pokud v knihovnách dokumentů klepnete na název dokumentu, může ji aplikace sady Office neočekávaně otevřít jako soubor jen pro čtení. Chcete-li ji otevřít pro úpravy, klepněte na **otevřít** (tři tečky) a pak klepněte na **Upravit.**
