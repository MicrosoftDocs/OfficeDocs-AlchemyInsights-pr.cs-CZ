# <a name="set-up-one-monitor"></a>Nastavení jednoho monitoru

**Kontrola nastavení projektu počítače**

1. Otevřete nastavení projektu stisknutím následujících kláves současně: Klávesa Windows (tlačítko Start s logem Windows) + P.
2. V závislosti na předvolbách zvolte **Duplikovat**, **rozšířit**nebo **druhou obrazovku**.

**Poznámka:** Pokud má váš počítač pouze jeden zobrazovací port (například HDMI nebo VGA), budete potřebovat dokovací stanici nebo adaptér USB k nastavení více monitorů.

Pokud můžete nastavit jeden monitor, ale ne více, počítač nemusí být kompatibilní s více monitory. Další informace můžete vyhledat v původních dokumentech počítače.
