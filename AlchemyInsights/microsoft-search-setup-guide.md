# <a name="microsoft-search-setup-guide"></a>Průvodce nastavením služby Microsoft Search

K nastavení [a konfiguraci služby Microsoft](https://go.microsoft.com/fwlink/?linkid=2153798) Search použijte průvodce nastavením služby Microsoft Search. Nezáleží na tom, jestli ho chcete pilotně z pilotního projektu přenecháte skupině uživatelů nebo jestli ho chcete použít pro všechny uživatele ve vaší organizaci. Přiřadíte správce vyhledávání a editory vyhledávání a pak přizpůsobíte vyhledávání pro své uživatele odpověďmi a dalšími možnostmi, jako je přidání rozšíření Bingu do Chromu nebo nastavení Bingu jako výchozího vyhledávacího modulu.
