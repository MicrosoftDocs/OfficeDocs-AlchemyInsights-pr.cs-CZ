# <a name="a-user-has-two-mailboxes"></a>Uživatel má dvě poštovní schránky

Hybridní prostředí, které používají Azure Active Directory připojení (připojení zvukové poplašné zařízení) nebo DirSync může způsobit neúmyslně vytvořit dvě poštovní schránky uživatele: jeden v organizaci v prostorách a druhý v cloudu.

Chcete-li tento problém vyřešit, naleznete v tématu [Trvale vymazat předchozí poštovní schránka Info](https://blogs.technet.microsoft.com/exchange/2018/01/17/permanently-clear-previous-mailbox-info/).
