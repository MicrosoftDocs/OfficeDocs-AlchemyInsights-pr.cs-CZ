# <a name="view-your-users-microsoft-365-apps-activity"></a>Zobrazení aktivit uživatelů aplikace Microsoft 365

[Aktivační sestava Office](https://docs.microsoft.com/microsoft-365/admin/activity-reports/microsoft-office-activations?view=o365-worldwide) vám nabízí přehled, kteří uživatelé instalovali a aktivovali aplikace Microsoft 365 na aspoň jednom zařízení.

Pomocí [sestavy microsoft 365 Apps](https://docs.microsoft.com/microsoft-365/admin/activity-reports/microsoft365-apps-usage?view=o365-worldwide)můžete navíc zobrazit aktivitu aplikací Microsoft 365 pro vaše uživatele.
