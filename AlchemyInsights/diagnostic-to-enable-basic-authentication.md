# <a name="diagnostic-to-enable-basic-authentication-for-exchange-online-protocols"></a>Diagnostika pro povolení základního ověřování Exchange Online protokolů

Pomocí této diagnostiky můžete povolit základní ověřování pro protokoly Exchange Online, jako jsou protokoly POP3, IMAP4, protokol Exchange ActiveSync, Exchange Web Services, Offline adresář, MAPI, RPC a Vzdálený PowerShell, které microsoft nedávno zakázal pro vaši organizaci. 

Prostřednictvím Centra zpráv posíláme přímou komunikaci, aby tenantům věděli, kde jsou připravení vypnout základní ověřování ve svém prostředí z důvodu bez použití, což pomůže chránit jejich prostředí před souvisejícími bezpečnostními riziky.
