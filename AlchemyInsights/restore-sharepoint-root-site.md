# <a name="restore-the-sharepoint-root-site"></a>Obnovení sharepointového kořenového webu

Obnovte odstraněný kořenový web z nového Centra pro správu SharePointu. Po obnovení nahraďte místo toho web.

1. Přejděte na [stránku Odstraněné](https://admin.microsoft.com/sharepoint?page=recycleBin&modern=true) weby. 
2. Vyberte kořenový web, který chcete obnovit.
3. Klikněte na **Obnovit**.
4. [Nahraďte kořenový web.](https://docs.microsoft.com/sharepoint/troubleshoot/sites/url-that-resides-under-root-site-collection-is-broken)

Další informace najdete v článku [o modernizaci kořenového webu.](https://docs.microsoft.com/sharepoint/modern-root-site)
