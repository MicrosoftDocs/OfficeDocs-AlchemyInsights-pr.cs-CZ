# <a name="service-principal-key"></a>Hlavní klíč služby

Pokyny ke správě klíčových přihlašovacích údajů pro instanční objekt najdete v následujících článcích:

- [Get-AzureADServicePrincipalKeyCredential](https://docs.microsoft.com/powershell/module/azuread/get-azureadserviceprincipalkeycredential) -pro načtení klíčových přihlašovacích údajů pro instanční objekt pomocí PowerShellu.

- [New-AzureADServicePrincipalKeyCredential](https://docs.microsoft.com/powershell/module/azuread/new-azureadserviceprincipalkeycredential) -k vytvoření nového klíčového pověření pro instanční objekt pomocí prostředí PowerShell.

- [Remove-AzureADServicePrincipalKeyCredential](https://docs.microsoft.com/powershell/module/azuread/remove-azureadserviceprincipalkeycredential) – Chcete-li odebrat přihlašovací údaje z hlavního objektu služby pomocí PowerShellu, odeberte.
