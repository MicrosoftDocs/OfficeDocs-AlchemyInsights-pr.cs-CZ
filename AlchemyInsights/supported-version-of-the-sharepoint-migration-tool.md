# <a name="supported-version-of-the-sharepoint-migration-tool"></a>Podporovaná verze nástroj pro přenesení služby SharePoint



Nástroj pro přenesení služby SharePoint aktuálně podporuje pouze přenesení ze služby SharePoint 2013. V současné době nejsou podporovány verze 2010 a 2016.
  
Další informace naleznete v tématu [zavedení SharePoint Migration Tool](https://go.microsoft.com/fwlink/?linkid=2044765&amp;clcid=0x409).
