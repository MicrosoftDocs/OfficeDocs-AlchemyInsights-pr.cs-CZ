# <a name="supported-version-of-the-sharepoint-migration-tool"></a>Podporovaná verze nástroje pro migraci SharePointu



Nástroj pro migraci serveru SharePoint podporuje migraci pouze z SharePointu 2013. Verze 2010 a 2016 nejsou v současné době podporované.
  
Další informace najdete v tématu [představení nástroje pro migraci SharePointu](https://go.microsoft.com/fwlink/?linkid=2044765&amp;clcid=0x409).
