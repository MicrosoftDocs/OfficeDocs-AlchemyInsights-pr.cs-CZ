# <a name="supported-version-of-the-sharepoint-migration-tool"></a>Podporovaná verze Nástroje pro migraci služby SharePoint



V současné době nástroj pro migraci SharePointu podporuje jenom migrace ze SharePointu 2013. Verze 2010 a 2016 nejsou v současné době podporovány.
  
Další informace naleznete [v tématu Introducing the SharePoint Migration Tool](https://go.microsoft.com/fwlink/?linkid=2044765&amp;clcid=0x409).
