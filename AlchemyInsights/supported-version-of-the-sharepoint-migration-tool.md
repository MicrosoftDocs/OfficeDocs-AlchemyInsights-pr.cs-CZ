# <a name="supported-version-of-the-sharepoint-migration-tool"></a>Podporovaná verze nástroje pro migraci služby SharePoint



V současnosti podporuje nástroj ADMT pouze migraci ze služby SharePoint 2013. Verze 2010 a 2016 nejsou v tomto okamžiku podporovány.
  
Další informace naleznete v tématu [představení nástroje pro migraci služby SharePoint](https://go.microsoft.com/fwlink/?linkid=2044765&amp;clcid=0x409).
