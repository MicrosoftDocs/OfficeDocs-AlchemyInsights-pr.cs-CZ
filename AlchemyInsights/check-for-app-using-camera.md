# <a name="check-for-app-using-camera"></a>Kontrola aplikace pomocí fotoaparátu

1. Vyberte **Start**a pak vyberte **Nastavení > ochrany osobních údajů > kamery** (MS-nastavení: Ochrana osobních údajů – webová kamera)
2. V seznamu aplikací Najděte aplikaci pomocí "aktuálně používaného" a zavřete ji.
