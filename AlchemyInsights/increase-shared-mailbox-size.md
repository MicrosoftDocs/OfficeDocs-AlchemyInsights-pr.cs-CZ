# <a name="increase-the-size-of-a-shared-mailbox"></a>Zvětšit velikost sdílené poštovní schránky

Můžete zvětšit velikost sdílené poštovní schránky z výchozí velikosti 50 GB, 100 GB sdílené poštovní schránky přiřadíte licenci pro Exchange Online P2 (která je zahrnuta v rámci licence E3 a E5). Při přiřazení licence P2 k sdílené poštovní schránky v [aktivních uživatelů](https://portal.office.com/adminportal/home)velikost automaticky zvýší během 15 minut.
