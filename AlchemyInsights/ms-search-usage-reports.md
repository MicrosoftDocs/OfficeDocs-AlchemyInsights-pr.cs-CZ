# <a name="get-started-with-using-microsoft-search-usage-reports"></a>Začínáme s používáním sestav využití služby Microsoft Search

Sestavy využití služby Microsoft Search poskytují přehled o fungování vyhledávání ve vaší organizaci. Tyto přehledy vám pomůžou najít obsah a zpříjemní vyhledávání uživatelům ve vaší organizaci.

Další informace najdete v tématu [Sestavy využití vyhledávání microsoftu](https://go.microsoft.com/fwlink/?linkid=2152048).
