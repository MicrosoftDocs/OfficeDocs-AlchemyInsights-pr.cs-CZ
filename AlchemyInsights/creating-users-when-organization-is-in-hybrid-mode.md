# <a name="create-users-in-hybrid-deployments"></a>Vytváření uživatelů v hybridních nasazeních

Pokud je vaše organizace hybridní nasazení mezi místním prostředím a Microsoft 365, musíte vytvořit všechny objekty uživatelů v místní službě Active Directory. Vytvoříte objekty uživatelů v příslušné organizační jednotce (OU), aby byly synchronizované do vaší organizace.
