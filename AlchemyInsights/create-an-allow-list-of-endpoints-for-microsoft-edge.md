# <a name="create-an-allow-list-of-endpoints-for-microsoft-edge"></a>Vytvoření seznamu povolených koncových bodů pro Microsoft Edge

Abyste měli k dispozici kompletní prvky Microsoft Edge, musíte je připojit k Internetu. Pokud chcete, aby prohlížeč komunikoval přes brány firewall a další bezpečnostní mechanismy na internetu, musíte do [seznamu povolených koncových bodů](https://go.microsoft.com/fwlink/?linkid=2135054)přidat sadu adres URL domény.
