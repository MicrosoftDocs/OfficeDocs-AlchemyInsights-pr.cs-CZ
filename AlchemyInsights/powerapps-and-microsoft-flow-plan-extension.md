# <a name="powerapps-and-microsoft-flow-plan-extension"></a>Rozšíření plánu PowerApps a Microsoft Flow

Pokud máte v úmyslu požádat o rozšíření plánu pro **Microsoft Flow** nebo **PowerApps** a váš tenant už má aktivní uživatele jedné z následujících funkcí, není nutné podat žádost o podporu.

- Vytváření a publikování vlastních konektorů v PowerApps a Microsoft Flow
- Vlastní akce HTTP integrované do toku mimo SharePoint a OneDrive
- Integrace místních dat prostřednictvím místní brány dat

Zákazníci s aktivními uživateli těchto funkcí získají automatické rozšíření do 31.
