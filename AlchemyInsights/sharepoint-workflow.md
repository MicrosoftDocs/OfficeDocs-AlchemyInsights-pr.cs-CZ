# <a name="workflows-in-sharepoint"></a>Pracovní postupy služby SharePoint

<p>Pracovní postupy služby SharePoint jsou předem naprogramované miniaplikace, které zjednodušit a automatizovat celou řadu obchodních procesů. Pracovní postupy v rozsahu shromažďujete podpisy, zpětnou vazbu nebo schválení plánu nebo dokument, ke sledování aktuálního stavu rutinní postup.</p> <p><strong>Začínáme:</strong> Pokud jste&rsquo;re začali pracovních postupů služby SharePoint, naleznete v tématu <a href="https://support.office.com/en-us/article/introduction-to-sharepoint-workflow-07982276-54e8-4e17-8699-5056eff4d9e3">Úvod do pracovních postupů služby SharePoint</a> pro další informace.</p> <p><strong>Toku Microsoft:</strong> Zvažte použití Microsoft Flow místo klasických řešení pracovního postupu služby SharePoint. Další informace naleznete v tématu <a href="https://support.office.com/en-us/article/create-a-flow-for-a-list-or-library-in-sharepoint-online-or-onedrive-for-business-a9c3e03b-0654-46af-a254-20252e580d01">vytvořit tok dat pro seznam nebo knihovnu SharePoint Online nebo OneDrive</a>.</p> <p><strong>Pracovní postup řešení potíží a ladění:</strong> Ladění pracovních postupů se podobá předchozích verzí. Další informace naleznete v tématu <a href="https://docs.microsoft.com/en-us/sharepoint/dev/general-development/debugging-sharepoint-server-workflows">pracovní postupy ladění služby SharePoint</a>. Další informace o odstraňování potíží naleznete v tématu <a title="chyby ověření pracovního postupu odstraňování problémů serveru SharePoint Server" href="https://docs.microsoft.com/en-us/sharepoint/dev/general-development/troubleshooting-sharepoint-server-workflow-validation-errors-in-visio">Odstraňování chyby ověření pracovního postupu služby SharePoint Server</a>.&nbsp;</p>
