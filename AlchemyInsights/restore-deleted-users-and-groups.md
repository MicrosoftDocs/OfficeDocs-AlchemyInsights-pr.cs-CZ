# <a name="restore-deleted-users-and-groups"></a>Obnovení odstraněných uživatelů a skupin

Pokud chcete uživatele odstranit omylem, přečtěte si článek [obnovení nebo odebrání nedávno odstraněného uživatele pomocí služby Azure Active Directory](https://docs.microsoft.com/azure/active-directory/fundamentals/active-directory-users-restore?context=azure/active-directory/users-groups-roles/context/ugr-context).

Pokud chcete obnovit odstraněnou skupinu, přečtěte si článek [Obnovení odstraněné skupiny Microsoft 365 v Azure Active Directory](https://docs.microsoft.com/azure/active-directory/users-groups-roles/groups-restore-deleted).
