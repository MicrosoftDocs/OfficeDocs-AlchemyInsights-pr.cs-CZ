# <a name="onboard-a-windows-server-to-microsoft-defender-for-endpoint"></a>Připojení serveru s Windows k programu Microsoft Defender pro koncový bod

V operačním systému Windows Server nabízí Microsoft Defender pro koncový bod pokročilé funkce pro zjišťování a vyšetřování útoků. Tyto funkce jsou k dispozici bez problémů prostřednictvím konzoly Centrum zabezpečení v programu Microsoft Defender.

Defender for Endpoint podporuje následující verze Windows Serveru:

- Windows Server 2008 R2 SP1
- Windows Server 2012 R2
- Windows Server 2016
- Windows Server (SAC) verze 1803 a novější
- Windows Server 2019 a novější
- Základní edice Windows Serveru 2019

Další informace najdete v tématu [Připojení serverů Windows ke službě Microsoft Defender for Endpoint](https://go.microsoft.com/fwlink/?linkid=2143627).
