# <a name="use-microsoft-intune-to-configure-microsoft-edge-policy-settings-for-windows-10"></a>Nastavení zásad Microsoft Edge pro Windows 10 pomocí Microsoft Intune

Zásady a nastavení Microsoft Edge můžete nakonfigurovat přidáním konfiguračního profilu zařízení do Microsoft Intune. Jedná se o ekvivalent konfigurace zásad skupiny Microsoft Azure Active Directory nebo nastavení místního objektu zásad skupiny (GPO) na uživatelských zařízeních.

Další informace najdete v tématu [Vytvoření profilu pro správu nastavení v Microsoft Edge pro Windows 10](https://go.microsoft.com/fwlink/?linkid=2133700).

Tento článek se týká Microsoft Edge verze 77 a novějších verzí.
