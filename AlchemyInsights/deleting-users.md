# <a name="deleting-users"></a>Odstranění uživatelů

Pokud chcete okamžitě zablokovat přístup uživatelů a potřebujete zachovat jejich informace, přečtěte si přečtěte si informace [o blokování přístupu bývalého zaměstnance k datům Microsoftu 365](https://docs.microsoft.com/microsoft-365/admin/add-users/remove-former-employee#block-a-former-employees-access-to-microsoft-365-data).
  
Pokud chcete uživatele jednoduše odstranit a netýkáte se žádné z jeho dat, přečtěte si informace [o odstranění uživatele z vaší organizace](https://docs.microsoft.com/microsoft-365/admin/add-users/delete-a-user). Tím se okamžitě uvolní jejich licence.
