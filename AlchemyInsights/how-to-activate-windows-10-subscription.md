# <a name="activating-windows-10-subscriptions"></a>Aktivace předplatného systému Windows 10

Microsoft 365 Business, Microsoft 365 Education, Microsoft 365 a Enterprise poskytují digitální licence pro systém Windows 10 jako součást předplatného. Chcete-li předplatné aktivovat, musí již zařízení aktivovat systém Windows 10 pro (1703 nebo novější). Úplný seznam požadavků pro aktivaci předplatného systému Windows 10 naleznete v části požadavky na [aktivaci předplatného systému Windows 10](https://docs.microsoft.com/windows/deployment/windows-10-subscription-activation#requirements).
