# <a name="change-default-programs-in-windows-10"></a>Změna výchozích programů ve Windows 10

1. V nabídce **Start** vyberte **Nastavení**  >  **aplikace**–  >  **výchozí aplikace**.
2. Vyberte výchozí nastavení, které chcete nastavit, a pak zvolte aplikaci. V Microsoft Storu můžete taky získávat nové aplikace. Aplikace se dají nastavit jako výchozí.
3. Můžete chtít, aby se vaše soubory PDF, e-maily nebo hudba automaticky otevíraly přes jinou aplikaci, než kterou poskytuje Microsoft. Pokud chcete zvolit výchozí aplikace podle typu souboru, posuňte se dolů a vyberte  **zvolit výchozí aplikace podle typu souboru**.
