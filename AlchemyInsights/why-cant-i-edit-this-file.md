# <a name="why-cant-i-edit-this-file"></a>Proč tento soubor nemůžu upravovat?

Pokud je dokument, list nebo prezentace zamknuté pro úpravy a není možné je upravovat v desktopových aplikacích systému Windows, může se stát, že ho někdo upraví, nebo má soubor rezervovaný.

Podívejte se na některé z důvodů, [proč nemůžete soubor upravovat](https://support.office.com/article/why-can-t-i-edit-this-file-97315f48-aa5e-49d3-a4ae-a14b73daf87b).
