# <a name="why-cant-i-edit-this-file"></a>Proč nelze tento soubor upravit?

Pokud je dokument, list nebo prezentace uzamčena pro úpravy a nelze ji upravovat v desktopových aplikacích systému Windows, může se v něm jet, že soubor upravuje někdo jiný nebo že je soubor rezervován.

Podívejte se rychle na některé z [důvodů, proč pravděpodobně nebudete moci soubor upravit](https://support.office.com/article/why-can-t-i-edit-this-file-97315f48-aa5e-49d3-a4ae-a14b73daf87b).
