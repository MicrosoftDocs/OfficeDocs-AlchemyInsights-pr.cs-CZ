# <a name="use-jamf-to-configure-microsoft-edge-policy-settings-on-macos"></a>Konfigurace nastavení zásad Microsoft Edge v systému macOS pomocí funkce Jamf

Pokud chcete nakonfigurovat Microsoft Edge (stabilní kanál 81 nebo novější) v macOS, použijte soubor manifestu zásad Microsoft Edge na Jamf Pro 10.19.

Další informace najdete v tématu [Konfigurace nastavení zásad Microsoft Edge na macOS pomocí Jamf](https://go.microsoft.com/fwlink/?linkid=2134761).
