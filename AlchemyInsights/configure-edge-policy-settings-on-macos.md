# <a name="use-jamf-to-configure-microsoft-edge-policy-settings-on-macos"></a>Konfigurace nastavení zásad Microsoft Edge na macOS pomocí Jamf

Pokud chcete nakonfigurovat Microsoft Edge (stabilní kanál 81 nebo novější) na macOS, použijte soubor manifestu zásad Microsoft Edge na Jamf pro 10,19.

Další informace najdete v článku [o konfiguraci nastavení zásad Microsoft Edge na MacOS s Jamf](https://go.microsoft.com/fwlink/?linkid=2134761).
