# <a name="microsoft-viva-connections-availability"></a>Dostupnost aplikace Microsoft Viva Connections

Aplikace Microsoft Viva Connections bude pro desktop dostupná v první polovině roku 2021 a pro mobilní zařízení během léta 2021. V průběhu roku 2022 budou k dispozici další funkce.

Další informace najdete v článku Microsoft Viva Connections, který pomůže zajistit, aby se všichni [aktivně zapojili a byli informovaní.](https://techcommunity.microsoft.com/t5/microsoft-viva-blog/microsoft-viva-connections-helps-everyone-to-stay-engaged-and/ba-p/2107009)
