# <a name="outlook-error-cannot-start-microsoft-outlook"></a>Chyba aplikace Outlook: Nelze spustit aplikaci Microsoft Outlook

Máte potíže se spuštěním Outlooku 2016, Outlooku 2013 nebo Outlooku 2010 nebo se přijetím "Nelze spustit aplikaci Microsoft Outlook" Nelze otevřít chybové zprávy okna aplikace Outlook?
  
Nejprve použijte kroky řešení potíží v [tomto článku](https://support.office.com/article/I-can-t-start-Microsoft-Outlook-2016-2013-or-2010-or-receive-the-error-Cannot-start-Microsoft-Office-Outlook-Cannot-open-the-Outlook-Window-d1f69da6-b333-4650-97bf-4d77bd7abb85) k vyřešení těchto problémů se spuštěním. 
  
Můžete také spustit [nástroj Pokročilá diagnostika](https://aka.ms/SaRA-OutlookAdvDiagnostics) v postiženém počítači.
