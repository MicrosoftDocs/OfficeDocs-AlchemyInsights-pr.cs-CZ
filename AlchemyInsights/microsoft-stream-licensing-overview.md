# <a name="microsoft-stream-licensing-overview"></a>Přehled licencování Microsoft Stream

Microsoft Stream je cloudová služba licencovaná pro předplatné jednotlivých uživatelů. Je k dispozici jako součást oprávněných předplatných Microsoft 365. Informace o každém typu předplatného najdete v článku cena Microsoft 365.

- [Přehled licencování Microsoft Stream](https://docs.microsoft.com/stream/license-overview)
