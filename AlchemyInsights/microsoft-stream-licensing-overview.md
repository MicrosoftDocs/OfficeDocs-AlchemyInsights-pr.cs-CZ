# <a name="microsoft-stream-licensing-overview"></a>Microsoft Stream Licensing – přehled

Microsoft Stream je cloudová služba licencovaná na základě předplatného pro jednotlivé uživatele. Je k dispozici jako součást oprávněných předplatných Microsoft 365. Informace o jednotlivých typech předplatného najdete v cenách Microsoftu 365.

- [Microsoft Stream Licensing – přehled](https://docs.microsoft.com/stream/license-overview)
