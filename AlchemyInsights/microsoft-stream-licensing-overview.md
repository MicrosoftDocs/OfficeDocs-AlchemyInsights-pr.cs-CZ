# <a name="microsoft-stream-licensing-overview"></a>Přehled licencování Microsoft Streamu

Microsoft Stream je cloudová služba licencovaná na základě předplatného pro jednoho uživatele. Je k dispozici jako součást oprávněných předplatných Microsoft 365. Informace o jednotlivých typech předplatného najdete v článku o cenách Microsoftu 365.

- [Přehled licencování Microsoft Streamu](https://docs.microsoft.com/stream/license-overview)
