# <a name="proxy-address-error-while-creating-a-shared-mailbox"></a>Chyba adresy proxy serveru při vytváření sdílené poštovní schránky

Pokud jste se pokusili vytvořit sdílenou poštovní schránku a dostali tuto chybu "adresa proxy" SMTP:alias@domain.com "se už používá...", e-mailová adresa, kterou jste si zvolili, už převzala jiná poštovní schránka ve vaší organizaci.
  
Potřebujete najít uživatele, skupinu nebo sdílenou poštovní schránku s touto e-mailovou adresou a odstranit ji nebo změnit její e-mailovou adresu. Potom můžete vytvořit sdílenou poštovní schránku s uvolněnou e-mailovou adresou. Najděte ho na domovské stránce.
  
Pokud nechcete existující e-mailovou adresu odstranit, zvolte pro sdílenou poštovní schránku novou e-mailovou adresu.
