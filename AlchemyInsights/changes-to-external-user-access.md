# <a name="changes-to-external-user-access"></a>Změny pro přístup externích uživatelů

Od 23. března 2018 externí uživatelé nebudou nadále přístup k obsahu, která byla sdílena s všechny All Authenticated Users a všichni uživatelé formuláře, i když byl pozván nebo přidány do skupiny oprávnění. Externí uživatelé pouze budou moci získat přístup k obsahu, která byla sdílena s uživatele nebo skupiny, do kterých uživatel patří. Pokud chcete zachovat stávající nastavení, naleznete v tématu:https://support.microsoft.com/help/4089534/how-to-grant-the-everyone-claim-to-external-users-in-office-365
