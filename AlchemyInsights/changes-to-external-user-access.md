# <a name="changes-to-external-user-access"></a>Změny přístupu externího uživatele

března 2018 už externí uživatelé nebudou mít přístup k obsahu, který byl sdílen se všemi uživateli, se všemi ověřenými uživateli nebo se všemi uživateli formulářů, a to ani v případě, že byli pozváni nebo přidáni do autorizovaných skupin. Externí uživatelé budou mít přístup pouze k obsahu, který byl sdílen s uživatelem nebo se skupinami, do kterých uživatel patří. Pokud si chcete zachovat stávající nastavení, přečtěte si prosím:https://support.microsoft.com/help/4089534/how-to-grant-the-everyone-claim-to-external-users-in-office-365
