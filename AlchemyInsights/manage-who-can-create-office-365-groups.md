# <a name="manage-who-can-create-microsoft-365-groups"></a>Správa uživatelů, kteří mohou vytvářet skupiny Microsoft 365

Ve výchozím nastavení může skupiny O365 vytvářet kdokoliv z vaší organizace.
  
Pokud to chcete změnit, můžete vytvořit skupinu zabezpečení s PowerShellem, která omezuje vytváření skupin na podmnožinu uživatelů.
  
Postupujte podle podrobných pokynů tady: [Správa, kdo může vytvářet skupiny Microsoft 365](https://docs.microsoft.com/microsoft-365/admin/create-groups/manage-creation-of-groups)
