# <a name="report-on-file-and-folder-sharing-in-a-sharepoint-site"></a>Sestava sdílení souborů a složek na sharepointovém webu

Můžete vytvořit soubor CSV se všemi jedinečnými soubory, uživateli, oprávněními a odkazy pro konkrétní sharepointový web nebo knihovnu OneDrivu. To vám pomůže pochopit, jak se sdílení používá a zda jsou některé soubory nebo složky sdíleny s hosty. Další informace najdete v tématu [Sestava sdílení souborů a složek na sharepointovém webu](https://docs.microsoft.com/sharepoint/sharing-reports).
