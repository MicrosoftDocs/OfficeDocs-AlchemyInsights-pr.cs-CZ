# <a name="report-on-file-and-folder-sharing-in-a-sharepoint-site"></a>Sestava sdílení souborů a složek na sharepointovém webu

Můžete vytvořit soubor CSV se všemi jedinečnými soubory, uživateli, oprávněními a odkazy pro konkrétní sharepointový web nebo knihovnu OneDrivu. Pomůže vám to pochopit, jak se sdílení využívá a jestli se některé soubory nebo složky sdílejí s hosty. Další informace najdete v tématu [Sestava sdílení souborů a složek na sharepointovém webu](https://docs.microsoft.com/sharepoint/sharing-reports).
