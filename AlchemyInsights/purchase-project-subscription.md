# <a name="purchase-project-subscription"></a>Nákup předplatného Projectu

Správci si můžou zakoupit předplatné Projectu pro svou organizaci v Centru pro **správu Microsoftu 365**.

1. Přejděte na stránku **Služby fakturace**  >  **[nákupu](https://admin.microsoft.com/AdminPortal/Home?adminportal=1&msCV=%2BbOQtMNsz0ei8f5z.0.36#/catalog)** a zobrazte dostupná předplatná k nákupu pro vaši organizaci.

2. Vyhledejte „Project“ a najděte si příslušná předplatná.

3. Vyberte si z nabízených předplatných Projectu.

Na stránce [Porovnání řešení pro řízení projektů](https://products.office.com/project/compare-microsoft-project-management-software?tab=1&OCID=AID2000748_SEM_5j2j5X4B&MarinID=5j2j5X4B|78821275986631|%2Bproject%20%2Bo365|bb|c||1261139959949905|kwd-78821311481635:loc-190&lnkd=Bing_O365SMB_App&msclkid=185eccc165db1d3da290924720afcaa4&ef_id=XoY8vgAAAUTu0Bj8:20200402200513:s) najdete další informace o rozdílech mezi předplatnými Project Plan 1, 3 a 5.
