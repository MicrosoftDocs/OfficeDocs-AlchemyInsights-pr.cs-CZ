# <a name="webpage-html-web-resources-limitations"></a>Webová stránka (HTML) webové prostředky omezení

* Webový prostředek HTML nesmí obsahovat jakýkoli kód, který musí být spuštěn na serveru. Stránky technologie ASP.NET nelze uložit jako webové prostředky HTML.

* Webové prostředky HTML lze přijímat pouze omezený počet parametrů řetězce dotazu. Další informace: [předat parametry webové prostředky HTML](https://docs.microsoft.com/dynamics365/customer-engagement/developer/webpage-html-web-resources#BKMK_PassingParametersToWebResources)
